## Descripción

El [IFC explorador](/Arch_IfcExplorer "Arch IfcExplorer") es una utilidad simple para explorar el contenido de un archivo [IFC](/Arch_IFC/es "Arch IFC/es"). Los archivos IFC son archivos de texto y, por lo tanto, son legibles en un editor de texto, pero la información está condensada y es difícil de navegar. Esta utilidad presenta al usuario exactamente el mismo contenido, pero mostrado en un método organizado y legible. La biblioteca de software "[IfcOpenShell](/IfcOpenShell/es "IfcOpenShell/es")" debe estar instalada para que esta utilidad funcione.

El propósito de este explorador es simplemente permitirte comprobar lo que realmente está escrito en un archivo IFC, en caso de que quieras verificar si los contenidos fueron importados o exportados correctamente hacia y desde una aplicación compatible con IFC como FreeCAD.
