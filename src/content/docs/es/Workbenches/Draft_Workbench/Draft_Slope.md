## Descripción

El comando ![](/images/Draft_Slope.svg) **Borrador Pendiente** inclina la [Borrador Líneas](/Draft_Line/es "Draft Line/es") o [Borrador Hilos](/Draft_Wire/es "Draft Wire/es") seleccionados aumentando, o disminuyendo, la coordenada Z de todos los puntos después del primero. También puede utilizarse para aplanar [Borrador Hilos](/Draft_Wire/es "Draft Wire/es"). Tenga en cuenta que la pendiente es relativa al plano XY definido por la Datos**Placement** de los objetos.

![](/images/Draft_Slope_example.png)

On the left a horizontal Draft Line. On the right the same line with a slope value of 1 (angle is 45°)

## Utilización

1. Seleccione una o más [Borrador Líneas](/Draft_Line/es "Draft Line/es")  y/o [Borrador Hilos](/Draft_Wire/es "Draft Wire/es").
2. Hay varias formas de invocar el comando:
   * Pulsar el ![](/images/Draft_Slope.svg) Borrador Pendiente.
   * Seleccione la opción **Modificación → ![](/images/Draft_Slope.svg) Fijar pendiente** del menú.
   * Seleccione la opción **Utilidades → ![](/images/Draft_Slope.svg) Fijar pendiente** opción del menú.
3. Introduzca un valor de **Indice**. `0` significa que cada segmento es horizontal, `0,5` significa que la altura delta de cada segmento es `0,5` veces su longitud, etc. El valor también puede ser negativo.
4. Pulse Enter o el botón OK para finalizar el comando.

## Guión

See also: [Autogenerated API documentation](https://freecad.github.io/SourceDoc/) and [FreeCAD Scripting Basics](/FreeCAD_Scripting_Basics "FreeCAD Scripting Basics").

There is no Python method to slope objects. To emulate the results of the Draft Slope command the `Points` property of wire objects has to be modified.
