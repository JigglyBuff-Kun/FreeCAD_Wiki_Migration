## Introducción

El **menú Macro** es uno de los 7 sub-menús en el [menú estándar](/Standard_Menu/es "Standard Menu/es"):

[Archivo](/Std_File_Menu/es "Std File Menu/es")

[Editar](/Std_Edit_Menu/es "Std Edit Menu/es")

[Ver](/Std_View_Menu/es "Std View Menu/es")

[Herramientas](/Std_Tools_Menu/es "Std Tools Menu/es")

Macro

[Ventanas](/Std_Windows_Menu/es "Std Windows Menu/es")

[Ayuda](/Std_Help_Menu/es "Std Help Menu/es")

```
El menú Macro proporciona herramientas relacionadas con la grabación y ejecución de macros.

```

## Herramientas

Las siguientes herramientas están disponibles en este menú:

- ![](/images/Std_DlgMacroRecord.svg) [Grabación de Macro...](/index.php?title=Std_DlgMacroRecord/es&action=edit&redlink=1 "Std DlgMacroRecord/es (page does not exist)"): Abre un cuadro de diálogo para iniciar una sesión de grabación de macros.

- ![](/images/Std_DlgMacroExecute.svg) [Macros...](/index.php?title=Std_DlgMacroExecute/es&action=edit&redlink=1 "Std DlgMacroExecute/es (page does not exist)"): Abre un cuadro de diálogo para administrar y ejecutar macros.

- ![](/images/Std_RecentMacros.svg) Macros recientes: muestra una lista de macros utilizadas recientemente, que se pueden llamar con una combinación de teclas, por ejemplo, Ctrl+Shift+1. Esto se puede configurar en [Preferences](/Preferences_Editor "Preferences Editor").

- ![](/images/Std_DlgMacroExecuteDirect.svg) [Ejecutar macro](/index.php?title=Std_DlgMacroExecuteDirect/es&action=edit&redlink=1 "Std DlgMacroExecuteDirect/es (page does not exist)"): Ejecuta la macro actualmente en el editor de Python.

* [Adjuntar al depurador remoto...](/index.php?title=Std_MacroAttachDebugger/es&action=edit&redlink=1 "Std MacroAttachDebugger/es (page does not exist)"): Adjunta FreeCAD a un depurador remoto.

- ![](/images/Std_MacroStartDebug.svg) [Depurar macro](/index.php?title=Std_MacroStartDebug/es&action=edit&redlink=1 "Std MacroStartDebug/es (page does not exist)"): Inicia una sesión de depuración para la macro en el editor de Python.

- ![](/images/Std_MacroStopDebug.svg) [Parar depuración](/index.php?title=Std_MacroStopDebug/es&action=edit&redlink=1 "Std MacroStopDebug/es (page does not exist)"): Detiene una sesión de depuración.

* [Pasar al siguiente](/index.php?title=Std_MacroStepOver/es&action=edit&redlink=1 "Std MacroStepOver/es (page does not exist)"): Pasa sobre el código.

* [Entrar en](/index.php?title=Std_MacroStepInto/es&action=edit&redlink=1 "Std MacroStepInto/es (page does not exist)"): Entra al código.

* [Conmutar punto de parada](/index.php?title=Std_ToggleBreakpoint/es&action=edit&redlink=1 "Std ToggleBreakpoint/es (page does not exist)"): Alterna un punto de interrupción.

## Herramientas obsoletas

- ![](/images/Std_MacroStopRecord.svg) [Detener la grabación de la macro](/index.php?title=Std_MacroStopRecord/es&action=edit&redlink=1 "Std MacroStopRecord/es (page does not exist)"): Detiene una sesión de grabación de macros. Integrado en [Std DlgMacroRecord](/index.php?title=Std_DlgMacroRecord/es&action=edit&redlink=1 "Std DlgMacroRecord/es (page does not exist)"). No disponible en 0.22 and above.
