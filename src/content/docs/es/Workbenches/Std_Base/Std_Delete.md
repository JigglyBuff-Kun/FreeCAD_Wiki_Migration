## Descriptivo

Este comando borra los objetos seleccionados. Si no está seleccionado ningún objeto, no hace nada.

## Use

Los objetos seleccionados se pueden eliminar del documento presionando DEL, desde el menú **Editar → Borrar**, o desde el menú contextual de la [vista en árbol](/Tree_view/es "Tree view/es") . Todas las operaciones de borrado se pueden deshacer y rehacer.

## Scripting

See also: [Autogenerated API documentation](https://freecad.github.io/SourceDoc/) and [FreeCAD Scripting Basics](/FreeCAD_Scripting_Basics "FreeCAD Scripting Basics").

To delete an object use the `removeObject` method of the document object.

```
import FreeCAD

FreeCAD.ActiveDocument.removeObject("myObjectName")

```
