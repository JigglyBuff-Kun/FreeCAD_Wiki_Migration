## Description

The ![](/images/Sketcher_CreateTriangle.svg) [Sketcher CreateTriangle](/Sketcher_CreateTriangle "Sketcher CreateTriangle") tool creates an equilateral triangle. [introduced in 1.0](/Release_notes_1.0 "Release notes 1.0"): This is the same tool as [Sketcher CreateRegularPolygon](/Sketcher_CreateRegularPolygon "Sketcher CreateRegularPolygon") but with the number of sides preset to 3.

![](/images/SketcherCreateTriangleExample.png)

## Usage

See also: [Drawing aids](/Sketcher_Workbench#Drawing_aids "Sketcher Workbench").

1. There are several ways to invoke the tool:
   * Press the ![](/images/Sketcher_CreateTriangle.svg) [Triangle](/Sketcher_CreateTriangle "Sketcher CreateTriangle") button.
   * Select the **Sketcher → Sketcher geometries → ![](/images/Sketcher_CreateTriangle.svg) Create equilateral triangle** option from the menu.
   * Use the keyboard shortcut: G then P, then 3.
2. For further steps see [Sketcher CreateRegularPolygon](/Sketcher_CreateRegularPolygon#Usage "Sketcher CreateRegularPolygon").
