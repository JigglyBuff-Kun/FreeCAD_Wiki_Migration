## Description

## Bemutatás

A kijelölt hálók normálisainak kiegyenlítése.

## Usage

## Használat

1. Jelöljön ki hálókat a fanézetben.
2. Válassza a  Hálók → ![](/images/Mesh_HarmonizeNormals.png) Normálisok kiegyenlítése menüpontot a felső menüből.

## Jegyzetek

* This command can produce a mesh with flipped normals. The [Mesh FlipNormals](/Mesh_FlipNormals "Mesh FlipNormals") command can be used to correct this.
* For a clear indication of the orientation of the faces of mesh objects make sure the Nézet**Lighting** property of the mesh objects is set to `One side`. The color of the back side of their faces will then depend on the backlight settings: **Edit → Preferences... → Display → 3D View → Backlight color - Intensity**. See: [Preferences Editor](/Preferences_Editor#3D_View "Preferences Editor").
