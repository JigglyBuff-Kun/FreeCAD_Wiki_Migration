### Introduction

This tutorial is meant for intermediate Fusion360 users looking to try FreeCAD. It is intended to help you get started with small and meduim projects, by teaching key differences between the two softwares and how to transfer your Fusion-knowledge. You will *not* learn basic modelling techniques or get a deep dive into specific tools, but an introduction into the FreeCAD workflow. If you have a project idea and know how to do it in Fusion, this tutorial should be all you need to get it done in FreeCAD.

**Check out the [Tutorials page](/Tutorials "Tutorials") for more information on specific tools or query the [Command Reference](/List_of_Commands "List of Commands").**
