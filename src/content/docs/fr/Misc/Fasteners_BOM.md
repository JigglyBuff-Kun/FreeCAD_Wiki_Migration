## Description

La commande ![](/images/Fasteners_BOM.svg) **Fasteners Nomenclature** crée une feuille de calcul avec une nomenclature pour les fixations du document.

![](/images/Fasteners_BOM_Example.png)

Un modèle avec une feuille de tableur Fasteners et sa nomenclature

## Utilisation

1. Il existe plusieurs façons de lancer la commande :
   * Appuyez sur le bouton ![](/images/Fasteners_BOM.svg) Générer une nomenclature.
   * Sélectionnez l'option **Fasteners → ![](/images/Fasteners_BOM.svg) Générer une nomenclature** dans le menu.
2. Une feuille de tableur de nomenclature Fasteners est créée.

## Remarques

* Une feuille de tableur Fasteners de nomenclature ne se met pas à jour si les fixations sont modifiées, ou si des fixations sont ajoutées ou supprimées. Dans ce cas, la commande doit être utilisée pour créer une nouvelle feuille de calcul.
* Une feuille de tableur Fasteners de nomenclature peut être exportée avec la commande [Spreadsheet Exporter](/Spreadsheet_Export/fr "Spreadsheet Export/fr").
