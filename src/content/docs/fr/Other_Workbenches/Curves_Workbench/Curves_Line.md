## Description

L'outil ![](/images/Curves_Line.svg) **Curves Ligne** crée une ligne paramétrique entre deux sommets.

## Utilisation

1. Sélectionnez deux sommets dans la [vue 3D](/3D_view/fr "3D view/fr").
2. Il y a plusieurs façons de lancer l'outil :
   * Appuyez sur le bouton ![](/images/Curves_Line.svg) Parametric line.
   * Sélectionnez l'option **Curves → ![](/images/Curves_Line.svg) Parametric line** du menu.
3. Un objet **Line** est créé.

## Propriétés

Voir aussi : [Éditeur de propriétés](/Property_editor/fr "Property editor/fr")

Un objet **Line** est dérivé d'un [App FeaturePython](/App_FeaturePython/fr "App FeaturePython/fr") et hérite de toutes ses propriétés. Il possède également les propriétés supplémentaires suivantes :

### Données

Line

* Données**Vertex1** (`LinkSub`) : le premier sommet référencé.
* Données**Vertex2** (`LinkSub`) : le deuxième sommet référencé.
