## Description

L'![](/images/Curves_ZebraTool.svg) outil Zebra crée une texture Zebra pour l'inspection de surface. Cet outil fait partie des [ateliers externes](/External_workbenches/fr "External workbenches/fr") appelé [Curves](/Curves_Workbench/fr "Curves Workbench/fr").

## Utilisation

1. Passer à l'atelier ![](/images/Curves_workbench_icon.svg) [Curves](/Curves_Workbench/fr "Curves Workbench/fr") (à installer à partir du ![](/images/Std_AddonMgr.svg) [gestionnaire des extensions](/Std_AddonMgr/fr "Std AddonMgr/fr") si ce n'est pas déjà fait)
2. Sélectionnez la surface à tester
3. Pour appeler la commande, effectuez l'une des opérations suivantes:
   * Appuyez sur le bouton ![](/images/Curves_ZebraTool.svg) Zebra texture for surface inspection dans la barre d'outils.
   * Utilisez le **Surfaces → Zebra Tool**
4. Modifiez le paramètre d'Outil Zebra aux conditions requises

## Propriétés

* Données**Black Stripes Width** : modifie la largeur des bandes noires.
* Données**Scale** : redimensionne les rayures zébrées.
* Données**Rotation** : fait pivoter les rayures zébrées.

## Remarques

* Cliquez sur le bouton Quit pour quitter l'outil Zebra.
