## Description

L'outil ![](/images/Curves_WaterlineCurves.svg) **Curves Courbes de niveau** crée des courbes de niveau sur les faces sélectionnées.

## Utilisation

1. Sélectionnez une ou plusieurs faces.
2. Il y a plusieurs façons de lancer l'outil :
   * Appuyez sur le bouton ![](/images/Curves_WaterlineCurves.svg) WaterLine.
   * Sélectionnez l'option **Surfaces → ![](/images/Curves_WaterlineCurves.svg) WaterLine** du menu.
3. Un objet Waterline est créé.

## Properties

See also: [Property editor](/Property_editor "Property editor").

A **Waterline** object is derived from a [Part Feature](/Part_Feature "Part Feature") object and inherits all its properties. It also has the following additional properties:

### Data

Settings

* Données**Direction** (`Vector`): Axis of the cutting planes
* Données**Number** (`Integer`): The number of waterlines

Source

* Données**Source** (`LinkSubList`): The source face or object
