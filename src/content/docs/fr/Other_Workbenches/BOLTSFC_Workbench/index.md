## Prérequis

Pour utiliser BOLTS for FreeCAD

* FreeCAD (<http://freecadweb.org/>)
* Python 3.x
* pyyaml (<http://pyyaml.org/>)
* importlib (<https://pypi.python.org/pypi/importlib/1.0.2>) (uniquement pour python 2.6)

est requis.
