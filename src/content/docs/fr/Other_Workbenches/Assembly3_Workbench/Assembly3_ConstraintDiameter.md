## Description

La commande ![](/images/Assembly_ConstraintDiameter.svg) Contrainte diamètre contraint le diamètre d'un cercle ou d'un arc en 2D réalisé avec les outils de l'![](/images/Workbench_Draft.svg) [atelier Draft](/Draft_Workbench/fr "Draft Workbench/fr") en relation avec un ![](/images/Assembly_Workplane.svg) plan de travail.

Pour définir le diamètre, modifiez le paramètre Diamètre dans le panneau des propriétés.

## Utilisation

1. Sélectionnez un **cercle** ou un **arc** en 2D.
2. Activez la commande ![](/images/Assembly_ConstraintDiameter.svg) Contrainte diamètre en utilisant:
   * Le bouton ![](/images/Assembly_ConstraintDiameter.svg) Diameter.
3. Définissez la valeur **Diamètre** dans le panneau des propriétés (de la contrainte).
4. Appuyez sur le ![](/images/Assembly3_workbench_icon.svg) [Solve constraints](/Assembly3_ResolveConstraints/fr "Assembly3 ResolveConstraints/fr") ou sur le bouton ![](/images/Assembly_QuickSolve.svg) [Quick solve](/Assembly3_QuickSolve/fr "Assembly3 QuickSolve/fr") pour recalculer.

:   :   (si ![](/images/Assembly_AutoRecompute.svg) [Auto recompute](/Assembly3_AutoRecompute/fr "Assembly3 AutoRecompute/fr") et ![](/images/Assembly_SmartRecompute.svg) [Smart recompute](/Assembly3_SmartRecompute/fr "Assembly3 SmartRecompute/fr") sont désactivés).
