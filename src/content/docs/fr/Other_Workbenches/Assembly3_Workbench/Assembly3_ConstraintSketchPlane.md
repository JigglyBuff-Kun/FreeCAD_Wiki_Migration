## Description

La commande ![](/images/Assembly_ConstraintSketchPlane.svg) Plan esquisse crée un "plan d'esquisse" comme conteneur pour des objets Draft.

Elle peut être utilisé pour créer une esquisse de structure pour la cinématique 2D.

## Utilisation

1. Créer un conteneur d'assemblage vide en appuyant sur ![](/images/Assembly_New_Assembly.svg) [Create assembly](/Assembly3_CreateAssembly/fr "Assembly3 CreateAssembly/fr")
2. Sélectionnez l'objet d'assemblage juste créé et cliquez sur ![](/images/Assembly_Add_Workplane.svg) [Add workplane](/Assembly3_AddWorkplane/fr "Assembly3 AddWorkplane/fr")
3. Appuyez sur le bouton ![](/images/Assembly_ConstraintSketchPlane.svg) Sketch plane
4. Passez à l'![](/images/Workbench_Draft.svg) [atelier Draft](/Draft_Workbench/fr "Draft Workbench/fr") pour ajouter du contenu.

:   Avec un assemblage existant non vide, les étapes 1 et 2 peuvent être sautées et tout élément de face planaire peut être utilisé comme plan de travail pour le plan d'esquisse.
