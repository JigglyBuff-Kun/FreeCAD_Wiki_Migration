## Description

La commande ![](/images/Assembly_AutoElementVis.svg) Visibilité automatique des éléments permet de mettre temporairement en évidence les **éléments** sélectionnés.

Lorsqu'elle est activée, le ou les éléments sélectionnés sont mis en évidence.   
 (La visibilité des systèmes de coordonnées des éléments (ECS) dépend de l'état de basculement de la commande ![](/images/Assembly_ShowElementCS.svg) [Show element coordinate system](/Assembly3_ShowElementCS/fr "Assembly3 ShowElementCS/fr"))

![](/images/Assembly3_AutoElementVisibility-01.png)

Élément mis en évidence d'une face avec le système de coordonnées des éléments visible

## Utilisation

1. Activez la commande ![](/images/Assembly_AutoElementVis.svg) **Visibilité automatique des éléments** en utilisant l'une des méthodes suivantes :
   * Le bouton ![](/images/Assembly_AutoElementVis.svg) Auto element visibility.
   * L'option de menu **Assembly3 → Element style → ![](/images/Assembly_AutoElementVis.svg) Auto element visibility**.
