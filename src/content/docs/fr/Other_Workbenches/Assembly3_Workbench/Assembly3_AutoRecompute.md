## Description

La commande ![](/images/Assembly_AutoRecompute.svg) Recomputation automatique est une commande à bascule qui permet à la commande ![](/images/Assembly3_workbench_icon.svg) [Résoudre les contraintes](/Assembly3_ResolveConstraints/fr "Assembly3 ResolveConstraints/fr") de s'exécuter automatiquement.

## Utilisation

1. Activez la commande ![](/images/Assembly_AutoRecompute.svg) **Recomputation automatique** en utilisant l'un des éléments suivants :
   * Le bouton ![](/images/Assembly_AutoRecompute.svg) Auto recompute.
   * L'option de menu **Assembly3 → ![](/images/Assembly_AutoRecompute.svg) Auto recompute**.
