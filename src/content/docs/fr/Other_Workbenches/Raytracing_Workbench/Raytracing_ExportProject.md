## Description

Exporte le projet de tracé de rayons sélectionné dans un fichier.

## Utilisation

1. Sélectionnez un projet de lancer de rayons dans la [Vue en arborescence](/Tree_view/fr "Tree view/fr").
2. Cliquez sur ![](/images/Raytracing_ExportProject.svg) ou choisir Raytracing → ![](/images/Raytracing_ExportProject.svg) Exporter le projet... dans le menu supérieur.
