## Description

La commande ![](/images/Lattice2_AttachablePlacement.svg) Lattice2 AttachablePlacement attribue un placement ou une poignée à un objet afin qu'il puisse plus tard être référencé par un emplacement.

## Utilisation

1. Basculez vers l'atelier externe ![](/images/Lattice2_workbench_icon.svg) [Lattice2](/Lattice2_Workbench/fr "Lattice2 Workbench/fr")
2. Appelez la commande Placement attaché de plusieurs manières:
   * Appuyez sur l'icône ![](/images/Lattice2_AttachablePlacement.svg) dans la barre d'outils Lattice2
   * Utilisez la **Lattice2 → Attached Placement → Attached Placement** dans la liste déroulante Lattice2
   * Utilisez le raccourci clavier {{{1}}} {{{1}}}
3. Choisissez les options du menu de dialogue
