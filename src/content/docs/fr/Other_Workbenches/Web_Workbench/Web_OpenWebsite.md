## Description

La commande ![](/images/Web_OpenWebsite.svg) **Web Ouvrir le site web** ouvre le site de [FreeCAD](https://freecadweb.org) dans l'onglet web en cours, ou dans un nouvel onglet si l'onglet courant n'est pas un onglet web.

## Utilisation

1. Appuyez sur le bouton ![](/images/Web_OpenWebsite.svg) Ouvrir le site web....
