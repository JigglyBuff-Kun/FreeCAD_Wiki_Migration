## Description

L'atelier Silk est un [atelier externe](/External_workbenches/fr "External workbenches/fr") qui crée des surfaces NURBS dans FreeCAD.

## Installation

L'installation recommandée se fait par le ![](/images/Std_AddonMgr.svg) [Gestionnaire des extensions](/Std_AddonMgr/fr "Std AddonMgr/fr") via **Outils → Gestionnaire des extensions**.
