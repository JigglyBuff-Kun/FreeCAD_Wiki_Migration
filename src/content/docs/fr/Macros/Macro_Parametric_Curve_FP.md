## Description

**Mise à jour : lors de la mise à jour de 0.2024.05.08b à 0.2025.01.28b, vous devez d'abord désinstaller l'ancienne version, puis installer la nouvelle version. N'exécutez plus le fichier .FCMacro après la mise à jour. Vous devez supprimer ce fichier et exécuter le fichier .py à partir de la version 0.2025.01.28.**

Cette macro est une mise à jour de la [macro 3D Parametric Curve](/Macro_3D_Parametric_Curve/fr "Macro 3D Parametric Curve/fr") de Gomez Lucio, puis modifiée par Laurent Despeyroux le 9 février 2015. La macro a été mise à jour en un objet paramétrique Feature Python. Elle supporte les mêmes paramètres a, b, c, mais peut aussi avoir autant de paramètres d sous forme de d1, d2, d3, d4, etc. que vous le souhaitez. Si vous souhaitez référencer un [jeu de variables](/Std_VarSet/fr "Std VarSet/fr") ou un [objet DynamicData dd](/DynamicData_Workbench/fr "DynamicData Workbench/fr") dans une formule, vous pouvez utiliser la commande `fc(expr)` pour le faire. Par exemple, s'il existe une valeur flottante dans un objet dd nommé my_float et que vous souhaitez la référencer dans la formule pour le paramètre b, entrez pour b : `fc(dd.my_float)` ou si vous souhaitez l'utiliser de manière plus complexe : b : `fc(dd.my_float) * a + pi` comme autre exemple.

Il prend également en charge l'enregistrement des formules dans un fichier texte au format JSON et l'intégration de la formule en cours dans une feuille de calcul. La documentation complète est disponible sur github [Parametric_Curve_FP](https://github.com/mwganson/Parametric_Curve_FP).
