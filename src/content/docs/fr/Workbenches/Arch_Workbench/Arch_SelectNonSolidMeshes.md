## Description

L'outil **Arch Sélection de maillages non-manifold** identifie et sélectionne les maillages non solides (non-[manifold](http://fr.wikipedia.org/wiki/Vari%C3%A9t%C3%A9_%28g%C3%A9om%C3%A9trie%29)) dans un groupe sélectionné d'objets ![Mesh Workbench](/images/Workbench_Mesh.svg) [Mesh](/Mesh_Workbench/fr "Mesh Workbench/fr").

## Utilisation

1. Sélectionnez un objet mesh.
2. Sélectionnez l'option **Utilitaires → ![](/images/Arch_SelectNonSolidMeshes.svg) Sélectionner les maillages non-manifold** du menu.
