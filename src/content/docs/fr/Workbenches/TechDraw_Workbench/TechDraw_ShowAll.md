## Description

L'outil **TechDraw Montrer tout** outil permet d'afficher temporairement, puis de masquer, des lignes invisibles dans une vue. Les lignes peuvent être rendues invisibles avec l'outil [TechDraw Apparence des lignes](/TechDraw_DecorateLine/fr "TechDraw DecorateLine/fr"). Notez que "invisible" est un état cosmétique, à ne pas confondre avec les lignes cachées qui sont des constructions géométriques.

## Utilisation

1. Sélectionnez une vue avec des lignes invisibles sur une page ou dans la [vue en arborescence](/Tree_view/fr "Tree view/fr").
2. Il y a plusieurs façons de lancer l'outil :
   * Appuyez sur le bouton ![](/images/TechDraw_ShowAll.svg) Afficher/masquer les arrêtes invisibles.
   * Sélectionnez l'option **TechDraw → Ajouter des lignes → ![](/images/TechDraw_ShowAll.svg) Afficher/masquer les arrêtes invisibles** du menu.
3. Toutes les lignes invisibles de la vue sont affichées ou cachées.

## Remarques

* Pour rendre les lignes invisibles visibles en permanence, utilisez ![](/images/TechDraw_DecorateLine.svg). [TechDraw Apparence des lignes](/TechDraw_DecorateLine/fr "TechDraw DecorateLine/fr").

## Script

Voir aussi : [Autogenerated API documentation](https://freecad.github.io/SourceDoc/) et [FreeCAD Débuter avec les scripts](/FreeCAD_Scripting_Basics/fr "FreeCAD Scripting Basics/fr").

Les effets de l'outil Montrer tout peut être dupliqué dans la [macro](/Macros/fr "Macros/fr") ou la console [Python](/Python/fr "Python/fr").

```
v = App.ActiveDocument.View
vvo = v.ViewObject
vvo.ShowAllEdges = True
App.ActiveDocument.recompute()

```
