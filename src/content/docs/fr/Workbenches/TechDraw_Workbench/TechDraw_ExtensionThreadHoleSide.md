## Description

L'outil **TechDraw Corps de taraudage** ajoute un taraudage cosmétique à la vue latérale d'un perçage.

![](/images/TechDraw_ExtensionThreadHoleSideExample.png)

À droite, un perçage avec un corps de taraudage.

## Utilisation

1. Spécifiez à votre convenance les attributs de ligne avec l'outil ![](/images/TechDraw_ExtensionSelectLineAttributes.svg) [TechDraw Choix des attributs](/TechDraw_ExtensionSelectLineAttributes/fr "TechDraw ExtensionSelectLineAttributes/fr").
2. Sélectionnez deux lignes parallèles.
3. Il existe plusieurs façons de lancer l'outil :
   * Appuyez sur le bouton ![](/images/TechDraw_ExtensionThreadHoleSide.svg) Corps de taraudage.
   * Sélectionnez l'option **TechDraw → Extensions : taraudages/filetages → ![](/images/TechDraw_ExtensionThreadHoleSide.svg) Corps de taraudage** à partir du menu.
4. Un filet cosmétique est ajouté.
