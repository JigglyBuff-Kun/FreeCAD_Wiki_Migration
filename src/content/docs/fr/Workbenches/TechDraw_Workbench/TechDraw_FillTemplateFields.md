## Description

L'outil ![](/images/TechDraw_FillTemplateFields.svg)Remplir les champs du modèle remplit les champs de texte modifiables dans le bloc titre de la page en cours.

## Utilisation

1. Il y a plusieurs façons de lancer l'outil :
   * Appuyez sur le ![](/images/TechDraw_FillTemplateFields.svg) Mettre à jour les champs du modèle.
   * Sélectionnez l'option **TechDraw → Page → ![](/images/TechDraw_FillTemplateFields.svg) Mettre à jour les champs du modèle** du menu.
2. ...

## Remarques

Cet outil semble n'avoir aucun rapport avec le nouvel attribut **freecad:autofill** de l'espace de noms SVG de FreeCAD qui est apparu à peu près au même moment.
