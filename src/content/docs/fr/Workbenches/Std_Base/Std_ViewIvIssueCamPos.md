## Description

La commande **Std Position de la caméra** affiche les paramètres de la caméra de la [vue 3D](/3D_view/fr "3D view/fr") active dans la [vue rapport](/Report_view/fr "Report view/fr") et la [console Python](/Python_console/fr "Python console/fr").

```
OrthographicCamera {   viewportMapping ADJUST_CAMERA   position 57.73505 -57.73502 57.735027   orientation 0.74290609 0.30772209 0.59447283  1.2171158   nearDistance 81.588844   farDistance 109.60551   aspectRatio 1   focalDistance 100   height 100  }

```

Exemple de résultats : les paramètres de la caméra après être passés par une [vue isométrique](/Std_ViewIsometric/fr "Std ViewIsometric/fr") dans un nouveau document

## Utilisation

1. Sélectionnez l'option **Affichage → Stéréo → ![](/images/Std_ViewIvIssueCamPos.svg) Récupérer la position de la caméra** du menu.

## Remarques

* Les paramètres de la caméra peuvent être utilisés pour ajouter des vues figées à un fichier \*.cam. Voir [Std Figer l'affichage](/Std_FreezeViews/fr "Std FreezeViews/fr").

## Script

Voir aussi : [Autogenerated API documentation](https://freecad.github.io/SourceDoc/) et [FreeCAD Débuter avec les scripts](/FreeCAD_Scripting_Basics/fr "FreeCAD Scripting Basics/fr").

La méthode `getCamera` de l'objet View renvoie les mêmes paramètres de la caméra dans une seule chaîne.

```
import FreeCADGui

view = FreeCADGui.ActiveDocument.ActiveView
view.getCamera()

```
