## Description

La commande **Std Réduire/développer** bascule du mode document de la [vue en arborescence](/Tree_view/fr "Tree view/fr") au mode réduction de document. Dans ce mode, l'activation de la [vue 3D](/3D_view/fr "3D view/fr") d'un document développera automatiquement ce document dans l'arborescence et réduira tous les autres documents. Les autres modes sont [Arborescence Document unique](/Std_TreeSingleDocument/fr "Std TreeSingleDocument/fr") et [Arborescence Tous les documents](/Std_TreeMultiDocument/fr "Std TreeMultiDocument/fr").

## Utilisation

1. Il existe plusieurs façons de lancer la commande :
   * Cliquez sur la flèche noire vers le bas à droite du bouton ![](/images/Std_TreeSyncView.svg) et sélectionnez l'option **Réduire/développer** dans le menu déroulant. Remarque : l'image du bouton changera en fonction de l'option sélectionnée.
   * Sélectionnez l'option **Affichage → Actions dans l'arborescence → Réduire/développer** du menu.
