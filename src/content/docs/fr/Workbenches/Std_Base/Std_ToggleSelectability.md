## Description

La commande **Std Basculer la sélectivité** bascule la sélectionnabilité des objets dans les [vues 3D](/3D_view/fr "3D view/fr").

## Utilisation

1. Sélectionnez un ou plusieurs objets.
2. Il existe plusieurs façons de lancer la commande :
   * Sélectionnez l'option **Affichage → Visibilité → ![](/images/Std_ToggleSelectability.svg) Activer/désactiver la sélectivité** du menu.
   * Sélectionnez l'option **![](/images/Std_ToggleSelectability.svg) Activer/désactiver la sélectivité** dans le menu contextuel de la [vue en arborescence](/Tree_view/fr "Tree view/fr") ou de la vue 3D.

## Remarques

* La sélectibilité d'un objet peut également être modifiée via sa propriété Données**Selectable** associée dans l'[éditeur de propriétés](/Property_editor/fr "Property editor/fr").

## Script

Voir aussi : [Autogenerated API documentation](https://freecad.github.io/SourceDoc/) et [FreeCAD Débuter avec les scripts](/FreeCAD_Scripting_Basics/fr "FreeCAD Scripting Basics/fr").

La propriété `Selectable` d'un objet détermine sa sélectibilité.

```
import FreeCADGui

obj = FreeCADGui.ActiveDocument.myObjectName

obj.Selectable = not obj.Selectable

```
