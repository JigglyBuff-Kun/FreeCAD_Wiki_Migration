## Description

La commande ![](/images/Draft_Heal.svg) **Draft Réparer** répare les objets [Draft](/Draft_Workbench/fr "Draft Workbench/fr") problématiques trouvés dans de très vieux fichiers. Elle essaie de recréer les anciens objets à partir de zéro et de transférer leurs propriétés aux nouveaux objets.

## Utilisation

1. Sélectionnez éventuellement un ou plusieurs objets problématiques. Si aucun objet n'est sélectionné, le document entier sera traité.
2. Sélectionnez l'option **Utilitaires → ![](/images/Draft_Heal.svg) Réparer** dans le menu.
3. Si aucune erreur n'est trouvée, la commande ne fera rien.
