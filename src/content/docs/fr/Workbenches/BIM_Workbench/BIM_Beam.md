## Description

L'outil **BIM Poutre** crée un objet **Poutre**, un élément de structure horizontal, entre deux points.

## Utilisation

1. Il y a plusieurs façons de lancer l'outil :
   * Appuyez sur le bouton ![](/images/BIM_Beam.svg) Poutre.
   * Sélectionnez l'option **3D/BIM → ![](/images/BIM_Beam.svg) Poutre** du menu.
   * Utilisez le raccourci clavier : B puis M.
