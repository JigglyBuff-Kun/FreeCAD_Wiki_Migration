## Description

La commande **Harmoniser les normales‎** harmonise les normales des objets maillés.

## Utilisation

1. Sélectionnez un ou plusieurs objets maillés.
2. Il existe plusieurs façons de lancer la commande :
   * Appuyez sur le bouton ![](/images/Mesh_HarmonizeNormals.svg) Harmoniser les normales
   * Sélectionnez l'option **Maillages → ![](/images/Mesh_HarmonizeNormals.svg) Harmoniser les normales‎** du menu.

## Remarques

* Cette commande peut produire un maillage avec des normales inversées. La commande [Mesh Inverser les normales](/Mesh_FlipNormals/fr "Mesh FlipNormals/fr") peut être utilisée pour corriger cela.
* Pour une indication claire de l'orientation des faces des objets maillés, assurez-vous que la propriété Vue**Lighting** des objets maillés est définie à `One side`. La couleur de l'arrière de leurs faces dépendra alors des paramètres de rétroéclairage : **Edition → Préférences... → Affichage → Vue 3D → Couleur du rétroéclairage - Intensité**. Voir : [Réglage des préférences](/Preferences_Editor/fr#Vue_3D "Preferences Editor/fr").
