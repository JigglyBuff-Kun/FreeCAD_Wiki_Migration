## Description

L'outil ![](/images/CAM_Comment.svg) Commentaire insère un commentaire. Lors de l'export d'un projet en G-code, le commentaire sera inclus dans le programme G-code et peut être lu par les personnes qui regardent le code. Certains contrôleurs de machine les affichent aussi sur leurs consoles.

## Utilisation

1. Pressez le bouton ![](/images/CAM_Comment.svg) Commenter ou utilisez le raccourci clavier P puis C.
2. Écrivez votre commentaire dans la zone **Commentaire**

## Propriétés

* Données**Comment** : le texte de votre commentaire
