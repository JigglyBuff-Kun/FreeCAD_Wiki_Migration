## Description

L'outil ![](/images/CAM_Stop.svg) Arrêter insère une commande d'arrêt (M1). Cette commande suspendra le programme en cours d'exécution sur le contrôleur CNC dans l'attente d'une nouvelle action de l'utilisateur.

**Remarque : cela n'arrête pas la broche.**

## Utilisation

1. Pressez le bouton ![](/images/CAM_Stop.svg) Arrêter

## Propriétés

* Données**Stop** : définit si l'arrêt est facultatif ou obligatoire.
