## Description

Insère un nouvel objet de trajectoire vide dans la scène.

## Utilisation

Cliquez sur ![](/images/Robot_CreateTrajectory.svg) ou choisissez  Robot → ![](/images/Robot_CreateTrajectory.svg) Créer une trajectoire dans le menu du haut.
