## Description

L'outil ![](/images/Assembly_CreateJointRevolute.svg) Assembly Liaison pivot crée une liaison pivot (également connu sous le nom de liaison articulé, ou charnière en abrégé) permettant la rotation autour d'un seul axe entre deux pièces sélectionnées.

## Utilisation

1. Vous pouvez sélectionner deux entités géométriques de deux formes différentes. Les autres sélections seront rejetées.
2. Il y a plusieurs façons de lancer l'outil :
   * Appuyez sur le bouton ![](/images/Assembly_CreateJointRevolute.svg) Créer une liaison pivot
   * Sélectionnez l'option **Assemblage → ![](/images/Assembly_CreateJointRevolute.svg) Créer une liaison pivot** du menu.
   * Utilisez le raccourci clavier : R.
3. La fenêtre de dialogue **Créer une liaison** s'ouvre dans le [panneau des tâches](/Task_panel/fr "Task panel/fr") avec la liste des entités présélectionnées.
4. Pour les étapes suivantes, voir [Assemblage Liaison fixe](/Assembly_CreateJointFixed/fr#Utilisation "Assembly CreateJointFixed/fr").

## Propriétés

Voir aussi : [Éditeur de propriétés](/Property_editor/fr "Property editor/fr")
