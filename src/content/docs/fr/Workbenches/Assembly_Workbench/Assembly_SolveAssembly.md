## Description

L'outil ![](/images/Assembly_SolveAssembly.svg) Assemblage Résoudre un assemblage résout l'assemblage actif.

## Utilisation

1. Il y a plusieurs façons de lancer l'outil :
   * Appuyez sur le bouton ![](/images/Assembly_SolveAssembly.svg) Résoudre un assemblage.
   * Sélectionnez l'option **Assemblage → ![](/images/Assembly_SolveAssembly.svg) Résoudre un assemblage** du menu.
   * Utilisez le raccourci clavier : Z.
