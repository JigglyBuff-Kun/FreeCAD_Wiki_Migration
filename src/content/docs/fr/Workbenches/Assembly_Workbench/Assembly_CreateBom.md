## Description

L'outil ![](/images/Assembly_CreateBom.svg) Assembly Nomenclature génère une nomenclature à partir d'un assemblage sélectionné ou à partir du document si aucun assemblage n'est sélectionné.

## Utilisation

1. Il y a plusieurs façons de lancer l'outil :
   * Appuyez sur le bouton ![](/images/Assembly_CreateBom.svg) Créer une nomenclature.
   * Sélectionnez l'option **Assemblage → ![](/images/Assembly_CreateBom.svg) Créer une nomenclature** du menu.
   * Utiliser le raccourci clavier : O.
