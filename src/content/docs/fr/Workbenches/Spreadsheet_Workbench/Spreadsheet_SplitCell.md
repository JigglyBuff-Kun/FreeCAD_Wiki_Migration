## Description

L'outil ![](/images/Spreadsheet_SplitCell.svg) **Spreadsheet Diviser une cellule** divise une cellule préalablement fusionnée.

## Utilisation

1. Assurez-vous qu'il existe une [feuille de calcul](/Spreadsheet_CreateSheet/fr "Spreadsheet CreateSheet/fr") active.
2. Sélectionnez une cellule [fusionnée](/Spreadsheet_MergeCells/fr "Spreadsheet MergeCells/fr").
3. Il y a plusieurs façons de lancer l'outil :
   * Appuyez sur le bouton ![](/images/Spreadsheet_SplitCell.svg) Diviser la cellule.
   * Sélectionnez l'option **Spreadsheet → ![](/images/Spreadsheet_SplitCell.svg) Diviser la cellule** du menu.
