## Description

Cet outil efface toutes les mesures de la [vue 3D](/3D_view/fr "3D view/fr").

## Utilisation

1. Il y a plusieurs façons de lancer la commande :
   * Appuyez sur le bouton ![](/images/Part_Measure_Clear_All.svg) Tout effacer.
   * Sélectionnez l'option **Mesure → ![](/images/Part_Measure_Clear_All.svg) Tout effacer**  dans le menu.
