## Description

Supprime tous les [Plans de coupe](/FEM_ClippingPlaneAdd/fr "FEM ClippingPlaneAdd/fr") existants.

## Utilisation

1. Il y a plusieurs façons de lancer la commande :
   * Appuyez sur le bouton ![](/images/FEM_ClippingPlaneRemoveAll.svg) Supprimer tous les plans de coupe.
   * Sélectionnez l'option **Utilitaires → ![](/images/FEM_ClippingPlaneRemoveAll.svg) Supprimer tous les plans de coupe** du menu.
