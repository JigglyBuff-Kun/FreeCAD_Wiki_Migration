## Description

The ![](/src/assets/images/Spreadsheet_AlignRight.svg) **Spreadsheet AlignRight** tool aligns the contents of selected cells to the right.

## Usage

1. Make sure there is an active [spreadsheet](/Spreadsheet_CreateSheet "Spreadsheet CreateSheet").
2. Select one or more cells.
3. There are several ways to invoke the tool:
   - Press the ![](/src/assets/images/Spreadsheet_AlignRight.svg) [Align right](/Spreadsheet_AlignRight "Spreadsheet AlignRight") button.
   - Select the **Spreadsheet → Alignment → ![](/src/assets/images/Spreadsheet_AlignRight.svg) Align right** option from the menu.
