## Description

The ![](/src/assets/images/Spreadsheet_AlignTop.svg) **Spreadsheet AlignTop** tool aligns the contents of selected cells to the top.

## Usage

1. Make sure there is an active [spreadsheet](/Spreadsheet_CreateSheet "Spreadsheet CreateSheet").
2. Select one or more cells.
3. There are several ways to invoke the tool:
   - Press the ![](/src/assets/images/Spreadsheet_AlignTop.svg) [Align top](/Spreadsheet_AlignTop "Spreadsheet AlignTop") button.
   - Select the **Spreadsheet → Alignment → ![](/src/assets/images/Spreadsheet_AlignTop.svg) Align top** option from the menu.
