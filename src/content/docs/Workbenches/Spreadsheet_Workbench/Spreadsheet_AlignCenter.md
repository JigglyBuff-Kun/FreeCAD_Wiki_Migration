## Description

The ![](/images/Spreadsheet_AlignCenter.svg) **Spreadsheet AlignCenter** tool aligns the contents of selected cells to the center horizontally.

## Usage

1. Make sure there is an active [spreadsheet](/Spreadsheet_CreateSheet "Spreadsheet CreateSheet").
2. Select one or more cells.
3. There are several ways to invoke the tool:
   - Press the ![](/images/Spreadsheet_AlignCenter.svg) [Align center](/Spreadsheet_AlignCenter "Spreadsheet AlignCenter") button.
   - Select the **Spreadsheet → Alignment → ![](/images/Spreadsheet_AlignCenter.svg) Align center** option from the menu.
