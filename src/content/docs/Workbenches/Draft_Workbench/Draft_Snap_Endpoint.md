## Description

The ![](/src/assets/images/Draft_Snap_Endpoint.svg) **Draft Snap Endpoint** option snaps to the endpoints of edges. The edges can belong to [Draft](/Draft_Workbench "Draft Workbench") or [BIM](/BIM_Workbench "BIM Workbench") objects but also to objects created with other [workbenches](/Workbenches "Workbenches").

![](/src/assets/images/Draft_Snap_Endpoint_example.png)

Snapping the second point of a line to the endpoint of an edge

## Usage

For general information about snapping see [Draft Snap](/Draft_Snap "Draft Snap").

1. Make sure snapping is enabled. See ![](/src/assets/images/Draft_Snap_Lock.svg) [Draft Snap Lock](/Draft_Snap_Lock "Draft Snap Lock").
2. If **Draft Snap Endpoint** is not active do one of the following:
   - Press the ![](/src/assets/images/Draft_Snap_Endpoint.svg) [Snap endpoint](/Draft_Snap_Endpoint "Draft Snap Endpoint") button in the Draft snap toolbar.
   - [Draft](/Draft_Workbench "Draft Workbench"): Hold down the ![](/src/assets/images/Draft_Snap_Lock.svg)![](/src/assets/images/Toolbar_flyout_arrow.svg) button in the [Draft snap widget](/Draft_snap_widget "Draft snap widget") and in the menu that opens select the **![](/src/assets/images/Draft_Snap_Endpoint.svg) Snap endpoint** option.
   - [BIM](/BIM_Workbench "BIM Workbench"): Select the **Snapping → ![](/src/assets/images/Draft_Snap_Endpoint.svg) Snap endpoint** option from the menu, or from the [3D view](/3D_view "3D view") context menu.
3. Choose a Draft or BIM command to create your geometry.
4. Note that you can also change snap options while a command is active.
5. Move the cursor over an edge.
6. The edge is highlighted.
7. If an endpoint is found the point is marked and the ![](/src/assets/images/Draft_Snap_Endpoint.svg) icon is displayed near the cursor.
8. Optionally move the cursor closer to the other endpoint to select that point instead.
9. Click to confirm the point.
