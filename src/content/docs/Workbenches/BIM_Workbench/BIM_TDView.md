## Description

The ![](/src/assets/images/BIM_TDView.svg) [BIM TDView](/BIM_TDView "BIM TDView") tool creates a new TechDraw view from a section plane or 2D objects.

## Usage

1. There are several ways to invoke the tool:
   - Press the ![](/src/assets/images/BIM_TDView.svg) [View](/BIM_TDView "BIM TDView") button.
   - Select the **Annotation → ![](/src/assets/images/BIM_TDView.svg) View** option from the menu.
