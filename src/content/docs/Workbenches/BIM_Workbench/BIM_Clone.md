## Description

The **BIM Clone** tool is a shortcut to the standard [Draft Clone](/Draft_Clone "Draft Clone") tool, but launches a [Draft Move](/Draft_Move "Draft Move") command just after cloning, thus allowing you to clone and place the cloned object somewhere else in the model in one go.
