## Description

The **Std ViewIvStereoOff** command switches off stereo view mode in the active [3D view](/3D_view "3D view").

## Usage

1. Select the **View → Stereo → ![](/src/assets/images/Std_ViewIvStereoOff.svg) Stereo Off** option from the menu.
