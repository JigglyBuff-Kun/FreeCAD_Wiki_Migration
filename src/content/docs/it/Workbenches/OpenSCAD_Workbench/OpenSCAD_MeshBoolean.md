## Description

## Descrizione

Crea un nuovo oggetto mesh con un'operazione booleana sulle forme.

## Usage

## Uso

1. Selezionare le forme nella vista ad albero.
2. Cliccare su ![](/images/OpenSCAD_MeshBoolean.png) o scegliere OpenSCAD → ![](/images/OpenSCAD_MeshBoolean.png) Mesh Booleana ... dal menu in alto.
3. Selezionare l'operazione da eseguire sulle forme.
4. Selezionare Esegui

- Viene creato un nuovo oggetto mesh

## Limitations

## Limitazioni

La funzione richiede che OpenSCAD sia installato e configurato nelle Modifica → Preferenze
