## 概要

The ![](/images/Sketcher_CreatePentagon.svg) [Sketcher CreatePentagon](/Sketcher_CreatePentagon "Sketcher CreatePentagon") tool creates a pentagon. [introduced in 1.0](/Release_notes_1.0 "Release notes 1.0"): This is the same tool as [Sketcher CreateRegularPolygon](/Sketcher_CreateRegularPolygon "Sketcher CreateRegularPolygon") but with the number of sides preset to 5.

![](/images/SketcherCreatePentagonExample.png)

## 使い方

See also: [Drawing aids](/Sketcher_Workbench#Drawing_aids "Sketcher Workbench").

1. There are several ways to invoke the tool:
   * Press the ![](/images/Sketcher_CreatePentagon.svg) [Pentagon](/Sketcher_CreatePentagon "Sketcher CreatePentagon") button.
   * Select the **Sketcher → Sketcher geometries → ![](/images/Sketcher_CreatePentagon.svg) Create pentagon** option from the menu.
   * Use the keyboard shortcut: G then P, then 5.
2. For further steps see [Sketcher CreateRegularPolygon](/Sketcher_CreateRegularPolygon#Usage "Sketcher CreateRegularPolygon").
