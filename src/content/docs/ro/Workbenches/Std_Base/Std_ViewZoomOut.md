## Description

## Descriere

Face să apară conținutul mai mic în vizualizarea 3D .

## Usage

## Cum se folosește

* Go to **View → Zoom‏‎ → ![](/images/Zoom-out.svg) Zoom Out** or press Ctrl + -.
* The view can also be zoomed out with the mouse scroll wheel.
