## Descriere

Acest instrument permite inspectarea dialectului de codul G intern FreeCAD pentru o operațiune a traiectoriei.

![](/images/Path_inspector.jpg)

## Utilizare

1. Select a Path object
2. Press the ![](/images/Path_Inspect.png) [Inspect G-code](/Path_Inspect "Path Inspect") button

## Opţiuni

- Alternatively, right-click a Path object → Inspect
- The font and colors of the editor window follow the settings from **Edit → Preferences → Editor**
