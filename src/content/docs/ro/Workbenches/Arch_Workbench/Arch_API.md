Funcțiile API care fac parte din atelierul Arch [Arch Workbench](/Arch_Workbench "Arch Workbench") pot fi utilizate atât în macro-uri[macros](/Macros "Macros") cât și din consola [Python](/Python "Python") după ce a fost importat modulul  `Arch`. Atelierul [Reinforcement Addon](/Reinforcement_Addon "Reinforcement Addon") are propriul [Reinforcement API](/Reinforcement_API "Reinforcement API").

Exempluː

```
import FreeCAD, Draft, Arch

p1 = FreeCAD.Vector(0, 0, 0)
p2 = FreeCAD.Vector(2000, 0, 0)
baseline = Draft.makeLine(p1, p2)

Arch.makeWall(baseline, length=None, width=200, height=2000)

```
