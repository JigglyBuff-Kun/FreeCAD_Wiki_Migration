## Description

## Descriere

Acest instrument permite selectarea conținutului unuia sau a mai multor grupuri selectate.

## Usage

## Cum se folosește

1. Selectați unul sau mai multe grupuri îîn vizualizarea arborescentă
2. Apăsați butonul ![](/images/Draft_SelectGroup.png) [Select group](/Draft_SelectGroup "Draft SelectGroup")
