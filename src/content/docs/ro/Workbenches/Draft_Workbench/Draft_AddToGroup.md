## Description

## Descriere

Acest instrument afișează un meniu rapid care vă permite să adăugați obiectele selectate la un grup deja existent, sau a le șterge din grupul lor curent

## Usage

## Cum se utilizează

1. Selectați obiectul(ele)
2. apăsați btonul ![](/images/Draft_AddToGroup.png) [Add to group](/Draft_AddToGroup "Draft AddToGroup")
