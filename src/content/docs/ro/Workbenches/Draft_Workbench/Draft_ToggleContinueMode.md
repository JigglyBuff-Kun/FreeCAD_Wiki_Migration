## Description

## Descriere

Unele instrumente ale modulelor Draft și Arch conțin o casetă de selectare Continue, care, atunci când este bifată, repornește comanda după ce ați terminat-o, deci nu este nevoie să apăsați din nou butonul de comandă și puteți crea rapid mai multe obiecte într-un rând, unul dupa altul. Această comandă activează/dezactivează această casetă de selectare pentru comenzile următoare.

## Usage

## Cum se folosește

1. Apăsați butonul ![](/images/Draft_ToggleContinueMode.png) [Toggle continue mode](/Draft_ToggleContinueMode "Draft ToggleContinueMode")
2. Porniți comanda care are o casetă Continue checkbox, ca de ex [Draft Line](/Draft_Line "Draft Line") sau [Arch Wall](/Arch_Wall "Arch Wall")
3. Abandonați comanda sau apăsați tasta ESC pentru a încheia atât comanda cât și modul continu.
