## Description

## Descriere

Acest instrument măsoară unghiul între două muchii drepte , două fațete planare sau muchii drepte, și afișează măsurătoarea din vizualizarea 3 D.

The appearance of the measurements can be changed in the [preferences](/PartDesign_Preferences#Measure "PartDesign Preferences").

![](/images/MeasureAngular1.PNG)

## Cum se folosește

1. Selectați două muchii drepte, două fațete plane sau cu o muchie dreaptă
2. Apăsați butonul ![](/images/Part_Measure_Angular.png) **Measure Angular‏‎** .
3. Alternativ comanda poate fi lansată fără o prealabilă selecție. Se deschide o casetă de dialog de selecție în Tab-ul Activități. Un widget de control oferă de asemenea butoane pentru a reseta selecția, a comuta afișarea măsurătorilor în vizualizarea 3D și pentru a șterge toate măsurătorile.
4. Măsurătorile sunt suprimate automat când este închis documentul.

## Notes

* You cannot use the [Draft](/Draft_Workbench "Draft Workbench") snap tools with this command.
* To add dimensions to drawings use the dimension tools from the [TechDraw Workbench](/TechDraw_Workbench "TechDraw Workbench").
* For more comprehensive measuring tools, install the ![](/images/Manipulator_workbench_icon.svg) [Manipulator Workbench](/Manipulator_Workbench "Manipulator Workbench") (an [external workbench](/External_workbenches "External workbenches")).
