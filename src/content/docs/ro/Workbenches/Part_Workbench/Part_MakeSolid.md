## Description

## Introducere

Această comandă creează un solid dintr-un obiect tip formă.

This command is typically used as one of the steps to create a solid from a mesh. See [Part ShapeFromMesh](/Part_ShapeFromMesh#Usage "Part ShapeFromMesh") for more information.

## Usage

## Utilizare

1. Selectați forma obiect.
2. Selectați  Part → Convert to solid din meniul principal.
3. Un nou obiect solid este creat din obiectul tip formă.

.

## Notes

## Limitări

Nu va fi nici o analiză sau validare a obiectului de formă.
  
Este recomandabil să utilizați traseul  Part →  Refine Shape înaintea conversiei în solid.

## Properties

See also: [Property editor](/Property_editor "Property editor").

The created objects are [Part Feature](/Part_Feature "Part Feature") objects with no additional properties.
