## Description

## Introduction

Fuzionează două sau mai multe plase selectate.

## Usage

## Utilizare

1. Selectează plasele în vizualizarea arborescentă
2. Selecați Meshes → Merge din meniul principal.

Plasa rezultantă va fi creată ca un obiect nou separat.
