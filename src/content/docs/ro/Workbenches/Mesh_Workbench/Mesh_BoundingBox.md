## Description

## Introducere

Vă oferă coordonatele Min / Max ale unei casete de delimitare pentru o singură plasă.

## Usage

## Utilizare

1. Selectați plasa în vizualizarea arborescentă
2. Selectați Meshes → Analyze → Boundings Info... din meniul principal.
3. Un meniu contextual va afișa coordonatele Min/Max casetei de delimitare
