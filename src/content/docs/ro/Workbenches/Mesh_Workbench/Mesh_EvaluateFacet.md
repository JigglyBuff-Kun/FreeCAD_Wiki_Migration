## Description

## Introducere

Oferă informații despre fațete.

```
Mesh: Ellipsoid Facet 3631: Points: <1817, 1818, 1866>, Neighbours: <3534, 3632, 3630>
Triangle: <[1.964574, 0.047063, 0.748046], [1.937166, 0.062461, 0.992797], [1.964574, -0.047063, 0.748046]>

```

Example of the information displayed in the Report view

## Usage

## Utilizare

1. Selectați Meshes → Analyze → Face info din meniul principal.
2. Cursor se va transforma în pipetă.
3. Click pe fațeta de afișarea a indexului acesteia pe ecran.
4. Right-click în vizualizarea 3D-view și selectați "Leave info mode" pentru a încheia comanda.
