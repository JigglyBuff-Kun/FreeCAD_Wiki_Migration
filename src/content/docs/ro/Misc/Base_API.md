Modulul de bază este conținut în interiorul modulului FreeCAD și conține constructori pentru diferite tipuri de obiecte folosite în mod frecvent în FreeCAD.

![](/images/Class.png) **BoundBox(***[Xmin,Ymin,Zmin,Xmax,Ymax,Zmax]***)**

![](/images/Class.png) **BoundBox(***Tuple, Tuple***)**

![](/images/Class.png) **BoundBox(***Vector, Vector***)**

**Description**:
Creează o casetă de restricționare.
O cutie de legare este un cub ortografic, care este o modalitate de a descrie granițele exterioare. Ai o casetă delimitată de la o mulțime de tipuri 3D. Este adesea folosit pentru a verifica dacă o entitate 3D se află în aria unui alt obiect. Verificând mai întâi interferența de limitare poate salva o mulțime de timp de calcul!

![](/images/Class.png) **Matrix(****)**

**Description**: Creates a 4x4 [Matrix](/Matrix_API "Matrix API"), care poate fi folosit pentru a aplica transformări în obiecte.

![](/images/Class.png) **Vector(****)**

![](/images/Class.png) **Vector(***x, y, z***)**

**Description**: Creates a FreeCAD 3D [Vector](/Vector_API "Vector API"), reprezentând un punct 3D sau o direcție.

![](/images/Class.png) **Placement(****)**

**Description**: Creates a [Placement](/Placement_API "Placement API").
