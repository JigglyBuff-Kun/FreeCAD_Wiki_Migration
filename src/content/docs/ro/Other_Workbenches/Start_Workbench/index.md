title: Atelierul/Panoul Start
---
:::caution
TheStart Workbenchis no longer included after version 0.21.It has been replaced by the Start page that can be accessed via theHelp menu.
:::
![](/images/Workbench_Start.svg)

Start workbench icon

[Start Workbench](/Start_Workbench "Start Workbench") nu este un atelier propriu zis, este doar pagina care este afișată atunci când deschideți FreeCAD fără a avea un document încărcat.

Din această interfață puteți vedea informații utile și puteți să accesați fișierele, exemplele, documentația și cele mai recente știri din lumea FreeCAD, deschise recent. Aveți posibilitatea să modificați modul în care se comportă Workbench Start sau să schimbați atelierul de lucru implicit în[Preferences Editor](/Preferences_Editor "Preferences Editor") în meniul **Edit → Preferences → Start**.
