## Описание

The ![](/images/Assembly_QuickSolve.svg) [Quick solve](/Assembly3_QuickSolve "Assembly3 QuickSolve") command starts the solver algorithm that relocates all objects of an assembly according to the constraints in the ![](/images/Assembly_Assembly_Constraints_Tree.svg) [Constraints container](/Assembly3_CreateAssembly#Description "Assembly3 CreateAssembly").

:   (Here's the space to put the differences to the ![](/images/Assembly3_workbench_icon.svg) [Resolve constraints](/Assembly3_ResolveConstraints "Assembly3 ResolveConstraints") command...)

## Применение

1. Activate the ![](/images/Assembly_QuickSolve.svg) **Quick solve** command using one of the following:
   * The ![](/images/Assembly_QuickSolve.svg) [Quick solve](/Assembly3_QuickSolve "Assembly3 QuickSolve") button.
   * The **Assembly3 → ![](/images/Assembly_QuickSolve.svg) Quick solve** menu option.
   * The keyboard shortcut: A then F.
