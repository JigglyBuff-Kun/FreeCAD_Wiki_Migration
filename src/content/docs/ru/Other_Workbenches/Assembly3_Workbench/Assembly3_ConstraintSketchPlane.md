## Описание

Команда ![](/images/Assembly_ConstraintSketchPlane.svg) Sketch plane создает плоскость эскиза в качестве контейнера для объектов Draft.

Его можно использовать для создания скелетного эскиза плоской кинематики.

## Применение

1. Создайте пустой контейнер сборки, нажав ![](/images/Assembly_New_Assembly.svg) [Create assembly](/Assembly3_CreateAssembly/ru "Assembly3 CreateAssembly/ru")
2. Выберите только что созданный объект сборки и нажмите ![](/images/Assembly_Add_Workplane.svg) [Add workplane](/index.php?title=Assembly3_AddWorkplane/ru&action=edit&redlink=1 "Assembly3 AddWorkplane/ru (page does not exist)")
3. Нажмите кнопку ![](/images/Assembly_ConstraintSketchPlane.svg) Sketch plane
4. Переключитесь на ![](/images/Workbench_Draft.svg) [Верстак Draft](/Draft_Workbench/ru "Draft Workbench/ru") для добавления контента

:   При существующей непустой сборке шаги 1 и 2 можно пропустить и любой элемент плоской грани можно использовать в качестве рабочей плоскости для плоскости эскиза.
