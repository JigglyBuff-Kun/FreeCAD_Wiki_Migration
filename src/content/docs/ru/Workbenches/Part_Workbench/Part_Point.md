## Описание

![](/images/Part_Point.svg) **Точка** — это параметрический примитив, который можно создать с помощью команды ![](/images/Part_Primitives.svg) [Примитивы...](/Part_Primitives/ru "Part Primitives/ru"). Её координаты относительны к системе координат, определенным в её свойстве Данные**Placement**.

## Применение

Смотри [Создать примитивы](/Part_Primitives#Usage/ru "Part Primitives").

## Свойства

Смотрите также: [Редактор свойств](/Property_editor/ru "Property editor/ru").

Объект Точка является производным от объекта [Конструктивный элемент](/Part_Feature/ru "Part Feature/ru") и наследует все его свойства. Он также имеет следующие дополнительные свойства:

### Данные

Вложение

Объект имеет те же свойства вложения, что и [Деталь Деталь2DОбъект](/Part_Part2DObject#Data/ru "Part Part2DObject").

Основные

* Данные**X** (`Расстояние`): Координата X точки. По умолчанию `0мм`.
* Данные**Y** (`Расстояние`): Координата Y точки. По умолчанию `0мм`.
* Данные**Z** (`Расстояние`): Координата Z точки. По умолчанию `0мм`.

## Написание скриптов

Смотри также: [Autogenerated API documentation](https://freecad.github.io/SourceDoc/), [Скрипты в верстаке Деталь](/Part_scripting/ru "Part scripting/ru") и [Основы работы со скриптами FreeCAD](/FreeCAD_Scripting_Basics/ru "FreeCAD Scripting Basics/ru").

Точка может быть создана с помощью метода `addObject()` документа:

```
point = FreeCAD.ActiveDocument.addObject("Part::Vertex", "myPoint")

```

* Где`"myPoint"` - это имя объекта.
* Функция возвращает только что созданный объект.

Пример:

```
import FreeCAD as App

doc = App.activeDocument()

point = doc.addObject("Part::Vertex", "myPoint")
point.X = 1
point.Y = 2
point.Z = 3

doc.recompute()

```
