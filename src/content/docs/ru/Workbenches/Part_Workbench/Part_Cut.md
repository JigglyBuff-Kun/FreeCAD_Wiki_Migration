## Описание

Инструмент ![](/images/Part_Cut.svg) **Обрезать** разрезает (вычитает) выбранные объекты деталей, причём последняя вычитается из первой. Эта операция полностью параметрическая, компоненты могут быть изменены, а результат пересчитан.

Этот инструмент представляет собой автоматизированную версию ![](/images/Part_Boolean.svg) [Булевы операции](/Part_Boolean/ru "Part Boolean/ru").

![](/images/Part_Cut_01.png)

## Применение

1. Выберите две фигуры.
2. Есть несколько способов вызвать инструмент:
   * Нажмите кнопку ![](/images/Part_Cut.svg) [Cut](/Part_Cut "Part Cut").
   * Выберите из меню **Деталь → Булевы операции → ![](/images/Part_Cut.svg) Обрезать** .

## Поддерживаемые исходные объекты

Входные объекты должны представлять собой фигуры [OpenCASCADE](/OpenCASCADE/ru "OpenCASCADE/ru"). Например, объекты, созданные с помощью верстаков Деталь (Part), Проектная деталь (PartDesign) или Эскизы (Sketcher). Для работы с сетками существуют специальные булевы инструменты в [верстаке Сетки (Mesh)](/Mesh_Workbench/ru "Mesh Workbench/ru").
