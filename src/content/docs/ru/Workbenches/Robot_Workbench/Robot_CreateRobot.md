## Описание

Добавляет в сцену робот (KUKA IR500).

## Применение

1. Нажмите на ![](/images/Robot_CreateRobot.svg) чтобы добавить робот KUKA IR500 в текущую сцену.
2. Добавить еще один робот или робот другой модели можно двумя способами через:
   * Пункт главного меню меню Робот → Вставить роботов.

   :   **ИЛИ**

   * Убедитесь, что документ в который вы хотите добавить робота создан и активен (сделать активным документ можно двойным щелчком в [древе проекта](/Tree_view/ru "Tree view/ru")).
   * Перейдите на вкладку «Задачи» в [комбо панели](/Tree_view/ru "Tree view/ru")).
3. Выберите одного из четырех предустановленных роботов.

## Ограничения

Для версий 0.15/0.16

* Задаются только роботы с числом осей до шести и с их вращательным движением.
* Роботы с поступательными движениями невозможны.
* Робот должен быть определен с помощью кода и VRML-файла, для получения дополнительной информации см. [6-ти осевой робот](/Robot_6-Axis/ru "Robot 6-Axis/ru").

## Примечания

По умолчанию доступны следующие модели роботов:

* KUKA IR500
* KUKA IR210
* KUKA IR125
* KUKA IR16
