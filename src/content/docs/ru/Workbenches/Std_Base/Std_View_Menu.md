## Введение

Меню "Вид" это одно из 7-ми меню [стандартного меню](/Standard_Menu/ru "Standard Menu/ru"):

[Файл](/Std_File_Menu/ru "Std File Menu/ru")

Вид

[Инструменты](/Std_Tools_Menu/ru "Std Tools Menu/ru")

[Макросы](/Std_Macro_Menu/ru "Std Macro Menu/ru")

[Окна](/Std_Windows_Menu/ru "Std Windows Menu/ru")

[Справка](/Std_Help_Menu/ru "Std Help Menu/ru")

Меню Вид предоставляет инструменты для настройки [3D вида](/3D_view/ru "3D view/ru"), свойств отображения объектов, а также инструменты, связанные с отображением компонентов интерфейса.

## Инструменты

В этом меню доступны следующие инструменты:

- ![](/images/Std_ViewCreate.svg) [Создать новый вид](/Std_ViewCreate/ru "Std ViewCreate/ru"): Создает новый вид.

- ![](/images/Std_OrthographicCamera.svg) [Ортогональная проекция](/Std_OrthographicCamera/ru "Std OrthographicCamera/ru"): Переключает вид в режим ортогональной проекции.

- ![](/images/Std_PerspectiveCamera.svg) [Перспективная проекция](/Std_PerspectiveCamera/ru "Std PerspectiveCamera/ru"): Переключает вид в режим перспективной проекции.

- ![](/images/Std_MainFullscreen.svg) [На весь экран](/Std_MainFullscreen/ru "Std MainFullscreen/ru"): Переключает главное окно в полноэкранный режим.

* Стандартные виды

  - ![](/images/Std_ViewFitAll.svg) [Уместить всё](/Std_ViewFitAll/ru "Std ViewFitAll/ru"): Уместить всё содержимое на экране.
  - ![](/images/Std_ViewFitSelection.svg) [Уместить выделенное](/Std_ViewFitSelection/ru "Std ViewFitSelection/ru"): Уместить выбранные объекты на экране.
  - Axonometric
    - ![](/images/Std_ViewIsometric.svg) [Изометрическая](/Std_ViewIsometric/ru "Std ViewIsometric/ru"): Установить изометрический вид.
    - ![](/images/Std_ViewDimetric.svg) [Диметрическая](/Std_ViewDimetric/ru "Std ViewDimetric/ru"): Установить диметрический вид.
    - ![](/images/Std_ViewTrimetric.svg) [Триметрическая](/Std_ViewTrimetric/ru "Std ViewTrimetric/ru"): Установить триметрический вид.
  - ![](/images/Std_ViewHome.svg) [Домой](/Std_ViewHome/ru "Std ViewHome/ru"): Установить вид по умолчанию.
  - ![](/images/Std_ViewFront.svg) [Спереди](/Std_ViewFront/ru "Std ViewFront/ru"): Установить вид спереди.
  - ![](/images/Std_ViewTop.svg) [Сверху](/Std_ViewTop/ru "Std ViewTop/ru"): Установить вид сверху.
  - ![](/images/Std_ViewRight.svg) [Справа](/Std_ViewRight/ru "Std ViewRight/ru"): Установить вид справа.
  - ![](/images/Std_ViewRear.svg) [Сзади](/Std_ViewRear/ru "Std ViewRear/ru"): Установить вид сзади.
  - ![](/images/Std_ViewBottom.svg) [Снизу](/Std_ViewBottom/ru "Std ViewBottom/ru"): Установить вид снизу.
  - ![](/images/Std_ViewLeft.svg) [Слева](/Std_ViewLeft/ru "Std ViewLeft/ru"): Установить вид слева.
  - ![](/images/Std_RotateLeft.png) [Повернуть влево](/Std_ViewRotateLeft/ru "Std ViewRotateLeft/ru"): Повернуть вид на 90 градусов против часовой стрелки.
  - ![](/images/Std_RotateRight.png) [Повернуть вправо](/Std_ViewRotateRight/ru "Std ViewRotateRight/ru"): Повернуть вид на 90 градусов по часовой стрелке.
  - [Сохранить рабочий вид](/Std_StoreWorkingView/ru "Std StoreWorkingView/ru"): Сохраняет настройки камеры активного 3D-вида. [представлено в версии 0.21](/Release_notes_0.21/ru "Release notes 0.21/ru")
  - [Восстановить рабочий вид](/Std_RecallWorkingView/ru "Std RecallWorkingView/ru"): Восстанавливает сохраненные настройки камеры активного 3D-вида. [представлено в версии 0.21](/Release_notes_0.21/ru "Release notes 0.21/ru")

* [Положения камеры](/Std_FreezeViews/ru "Std FreezeViews/ru")

  - [Сохранить виды в файл...](/Std_FreezeViews/ru#Сохранить_виды_в_файл... "Std FreezeViews/ru"): Сохраняет виды в файл.
  - [Загрузить виды из файла...](/Std_FreezeViews/ru#Загрузить_виды_из_файла... "Std FreezeViews/ru"): Загружает виды из файла.
  - [Запомнить вид](/Std_FreezeViews/ru#Запомнить_вид "Std FreezeViews/ru"): Запоминает текущий вид в сохраненом виде.
  - [Забыть все виды](/Std_FreezeViews/ru#Забыть_все_виды "Std FreezeViews/ru"): Удаляет все сохраненные виды.

* [Стиль представления](/Std_DrawStyle/ru "Std DrawStyle/ru")
  - ![](/images/Std_DrawStyleAsIs.svg) [Как есть](/Std_DrawStyle/ru#Как_есть "Std DrawStyle/ru"): Переключает стиль представления в режим: Как есть.
  - ![](/images/Std_DrawStylePoints.svg) [Точки](/Std_DrawStyle/ru#Точки "Std DrawStyle/ru"): Переключает стиль представления в режим: Точки.
  - ![](/images/Std_DrawStyleWireFrame.svg) [Каркас](/Std_DrawStyle/ru#Каркас "Std DrawStyle/ru"): Переключает стиль представления в режим: Каркас.
  - ![](/images/Std_DrawStyleHiddenLine.svg) [Скрытые линии](/Std_DrawStyle/ru#Скрытые_линии "Std DrawStyle/ru"): Переключает стиль представления в режим: Скрытые линии (wireframe).
  - ![](/images/Std_DrawStyleNoShading.svg) [Без затенения](/Std_DrawStyle/ru#Без_затенения "Std DrawStyle/ru"): Переключает стиль представления в режим: Без затенения.
  - ![](/images/Std_DrawStyleShaded.svg) [Только грани](/Std_DrawStyle/ru#Только_грани "Std DrawStyle/ru"): Переключает стиль представления в режим: Только грани.
  - ![](/images/Std_DrawStyleFlatLines.svg) [Плоские линии](/Std_DrawStyle/ru#Плоские_линии "Std DrawStyle/ru"): Переключает стиль представления в режим: Плоские линии.

- ![](/images/Std_SelBoundingBox.svg) [Bounding box](/Std_SelBoundingBox "Std SelBoundingBox"): Toggles the bounding box highlighting mode.

* Стерео

  - ![](/images/Std_ViewIvStereoRedGreen.svg) [Стерео красный/голубой](/Std_ViewIvStereoRedGreen/ru "Std ViewIvStereoRedGreen/ru"): Переключить режим стерео в красного/голубой режим.
  - ![](/images/Std_ViewIvStereoQuadBuff.svg) [Четверная буферизация стерео](/Std_ViewIvStereoQuadBuff/ru "Std ViewIvStereoQuadBuff/ru"): Переключить стерео в режим quad буффера.
  - ![](/images/Std_ViewIvStereoInterleavedRows.svg) [Стерео с чередованием строк](/Std_ViewIvStereoInterleavedRows/ru "Std ViewIvStereoInterleavedRows/ru"): Переключить стерео в режим чередования строк.
  - ![](/images/Std_ViewIvStereoInterleavedColumns.svg) [Стерео с чередованием столбцов](/Std_ViewIvStereoInterleavedColumns/ru "Std ViewIvStereoInterleavedColumns/ru"): Переключить стерео в режим чередования столбцов.
  - ![](/images/Std_ViewIvStereoOff.svg) [Выключить стерео](/Std_ViewIvStereoOff/ru "Std ViewIvStereoOff/ru"): Выключить стерео режим.
  - ![](/images/Std_ViewIvIssueCamPos.svg) [Выводить положение камеры](/Std_ViewIvIssueCamPos/ru "Std ViewIvIssueCamPos/ru"): вывести настройки камеры в [отчёт](/Report_view/ru "Report view/ru") и в [Python консоль](/Python_console/ru "Python console/ru").

* Масштаб

  - ![](/images/Std_ViewZoomIn.svg) [Увеличить](/Std_ViewZoomIn/ru "Std ViewZoomIn/ru"): Увеличивает масштаб изображения вида.
  - ![](/images/Std_ViewZoomOut.svg) [Уменьшить](/Std_ViewZoomOut/ru "Std ViewZoomOut/ru"):Уменьшает масштаб изображения вида.
  - ![](/images/Std_ViewBoxZoom.svg) [Увеличить область](/Std_ViewBoxZoom/ru "Std ViewBoxZoom/ru"): Масштабирует вид под размер прямоугольной области.

* Окно документа
  - [Закрепленное](/Std_ViewDockUndockFullscreen/ru#Docked "Std ViewDockUndockFullscreen/ru"): Встраивает 3D вид открепленный от основного окна приложения, обратно в приложение.
  - [Откреплённое](/Std_ViewDockUndockFullscreen/ru#Undocked "Std ViewDockUndockFullscreen/ru"): Отсоединяет 3D вид от основного окна приложения в виде отдельного окна.
  - ![](/images/Std_ViewFullscreen.svg) [На весь экран](/Std_ViewFullscreen/ru "Std ViewFullscreen/ru"): Переключает 3D вид в полноэкранный режим.

- ![](/images/Std_AxisCross.svg) [Показать/скрыть оси координат](/Std_AxisCross/ru "Std AxisCross/ru"): Переключает видимость осей координат.

- ![](/images/Std_ToggleClipPlane.svg) [Плоскость сечения](/Std_ToggleClipPlane/ru "Std ToggleClipPlane/ru"): Позволяет просмотреть объект в разрезе.

- ![](/images/Part_SectionCut.svg) [Persistent section cut](/Part_SectionCut "Part SectionCut"): Creates persistent cuts of objects and assemblies. _-- Available if the [Part Workbench](/Part_Workbench "Part Workbench") or the [PartDesign Workbench](/PartDesign_Workbench "PartDesign Workbench") has been loaded._

- ![](/images/Std_TextureMapping.svg) [Текстурирование...](/Std_TextureMapping/ru "Std TextureMapping/ru"): Временно накладывает текстуру поверх всех объектов.

* Видимость

  - ![](/images/Std_ToggleVisibility.svg) [Скрыть/показать выделенное](/Std_ToggleVisibility/ru "Std ToggleVisibility/ru"): Переключает видимость выбранных объектов.
  - ![](/images/Std_ShowSelection.svg) [Показать выделенное](/Std_ShowSelection/ru "Std ShowSelection/ru"): Показывает выбранные объекты.
  - ![](/images/Std_HideSelection.svg) [Скрыть выделенное](/Std_HideSelection/ru "Std HideSelection/ru"): Скрывает выбранные объекты.
  - ![](/images/Std_SelectVisibleObjects.svg) [Выбрать видимые объекты](/Std_SelectVisibleObjects/ru "Std SelectVisibleObjects/ru"): Выбрать все видимые объекты.
  - ![](/images/Std_ToggleObjects.svg) [Инвертировать все видимости](/Std_ToggleObjects/ru "Std ToggleObjects/ru"): Изменить видимость всех объектов на противоположную.
  - ![](/images/Std_ShowObjects.svg) [Показать все объекты](/Std_ShowObjects/ru "Std ShowObjects/ru"): Делает все объекты видимыми.
  - ![](/images/Std_HideObjects.svg) [Скрыть все объекты](/Std_HideObjects/ru "Std HideObjects/ru"): Скрывает все объекты.
  - ![](/images/Std_ToggleSelectability.svg) [Откл/вкл выделяемость](/Std_ToggleSelectability/ru "Std ToggleSelectability/ru"): Откл/вкл возможность выбирать объекты.
  - ![](/images/View_Measure_Toggle_All.svg) [Скрыть/показать всё измерения](/View_Measure_Toggle_All/ru "View Measure Toggle All/ru"): Откл/вкл отображение измерений деталей.
  - ![](/images/View_Measure_Clear_All.svg) [Удалить все измерения](/View_Measure_Clear_All/ru "View Measure Clear All/ru"): Удаляет все измерения детали.

* [Навигация/редактирование](/Std_ToggleNavigation/ru "Std ToggleNavigation/ru")

- ![](/images/Std_SetMaterial.svg) [Material...](/Std_SetMaterial "Std SetMaterial"): Sets the material of selected objects. _-- Available if the [Material Workbench](/Material_Workbench "Material Workbench") has been loaded directly or indirectly (via for example the [Part Workbench](/Part_Workbench "Part Workbench") or the [PartDesign Workbench](/PartDesign_Workbench "PartDesign Workbench"))._ [introduced in 1.0](/Release_notes_1.0 "Release notes 1.0")

- ![](/images/Std_SetAppearance.svg) [Внешний вид...](/Std_SetAppearance/ru "Std SetAppearance/ru")

- ![](/images/Std_RandomColor.svg) [Случайный цвет](/Std_RandomColor/ru "Std RandomColor/ru"): Устанавливает цвет фигуры случайным образом.

- ![](/images/Part_ColorPerFace.svg) [Color per face](/Part_ColorPerFace "Part ColorPerFace"): Sets the display properties of selected faces. _-- Available if the [Part Workbench](/Part_Workbench "Part Workbench") or the [PartDesign Workbench](/PartDesign_Workbench "PartDesign Workbench") is active._

- ![](/images/Std_ToggleTransparency.svg) [Toggle transparency](/Std_ToggleTransparency "Std ToggleTransparency"): Toggles the transparency of selected objects. [introduced in 1.0](/Release_notes_1.0 "Release notes 1.0")

* Dock window overlay [introduced in 1.0](/Release_notes_1.0 "Release notes 1.0")

  - [Toggle overlay for all](/index.php?title=Std_DockOverlayAll&action=edit&redlink=1 "Std DockOverlayAll (page does not exist)"): Toggles overlay mode for all docked windows.
  - [Toggle transparent for all](/index.php?title=Std_DockOverlayTransparentAll&action=edit&redlink=1 "Std DockOverlayTransparentAll (page does not exist)"): Toggles transparent mode for all docked overlay windows. This makes the docked windows stay transparent at all times.
  - [Toggle overlay](/index.php?title=Std_DockOverlayToggle&action=edit&redlink=1 "Std DockOverlayToggle (page does not exist)"): Toggles overlay mode for the docked window under the cursor.
  - [Toggle transparent](/index.php?title=Std_DockOverlayToggleTransparent&action=edit&redlink=1 "Std DockOverlayToggleTransparent (page does not exist)"): Toggles transparent mode for the docked window under cursor. This makes the docked window stay transparent at all times.
  - [Bypass mouse events in docked overlay windows](/index.php?title=Std_DockOverlayMouseTransparent&action=edit&redlink=1 "Std DockOverlayMouseTransparent (page does not exist)"): Bypasses all mouse events in docked overlay windows.
  - [Toggle left](/index.php?title=Std_DockOverlayToggleLeft&action=edit&redlink=1 "Std DockOverlayToggleLeft (page does not exist)"): Shows/hides the left overlay panel.
  - [Toggle right](/index.php?title=Std_DockOverlayToggleRight&action=edit&redlink=1 "Std DockOverlayToggleRight (page does not exist)"): Shows/hides the right overlay panel.
  - [Toggle top](/index.php?title=Std_DockOverlayToggleTop&action=edit&redlink=1 "Std DockOverlayToggleTop (page does not exist)"): Shows/hides the top overlay panel.
  - [Toggle bottom](/index.php?title=Std_DockOverlayToggleBottom&action=edit&redlink=1 "Std DockOverlayToggleBottom (page does not exist)"): Shows/hides the bottom overlay panel.

  - [Sync view](/Std_TreeSyncView "Std TreeSyncView"): Toggles the [Tree view](/Tree_view "Tree view") SyncView mode.
  - [Sync selection](/Std_TreeSyncSelection "Std TreeSyncSelection"): Toggles the Tree view SyncSelection mode.
  - [Sync placement](/Std_TreeSyncPlacement "Std TreeSyncPlacement"): Toggles the Tree view SyncPlacement mode.
  - [Pre-selection](/Std_TreePreSelection "Std TreePreSelection"): Toggles the Tree view PreSelection mode.
  - [Record selection](/Std_TreeRecordSelection "Std TreeRecordSelection"): Toggles the Tree view RecordSelection mode.
  - [Single document](/Std_TreeSingleDocument "Std TreeSingleDocument"): Switches the Tree view to SingleDocument mode.
  - [Multi document](/Std_TreeMultiDocument "Std TreeMultiDocument"): Switches the Tree view to MultiDocument mode.
  - [Collapse/Expand](/Std_TreeCollapseDocument "Std TreeCollapseDocument"): Switches the Tree view to CollapseDocument mode.
  - ![](/images/Std_TreeDrag.svg) [Initiate dragging](/Std_TreeDrag "Std TreeDrag"): Initiates a drag operation for selected objects in the Tree view.
  - ![](/images/Std_TreeSelection.svg) [Go to selection](/Std_TreeSelection "Std TreeSelection"): Scrolls the Tree view to the first created object in a 3D view selection.
  - ![](/images/Std_SelBack.svg) [Selection back](/Std_SelBack "Std SelBack"): Restores the previous Tree view selection.
  - ![](/images/Std_SelForward.svg) [Selection forward](/Std_SelForward "Std SelForward"): Restores the next Tree view selection.
