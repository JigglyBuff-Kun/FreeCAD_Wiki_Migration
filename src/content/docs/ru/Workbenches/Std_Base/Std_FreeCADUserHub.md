## Описание

Команда **Пользовательская документация** открывает wiki страницу пользовательской документации в интернет-браузере назначенном по умолчанию. Для работы команды требуется подключение к Интернет.

На текущий момент команда направляет пользователя на wiki страницу: <https://wiki.freecad.org/User_hub>.

## Применение

1. Выберите пункт меню **Помощь → ![](/images/Std_FreeCADUserHub.svg) Пользовательская документация**.
