## Описание

Команда **Вырезать** имеет ограничения: ее можно использовать только при работе с [ячейками электронных таблиц](/Spreadsheet_Workbench/ru "Spreadsheet Workbench/ru"). Команда копирует содержимое и свойства ячеек в буфер обмена, а затем очищает ячейки.

Чтобы вырезать объекты других типов, вы можете использовать команду [Копировать](/Std_Copy/ru "Std Copy/ru"), следом за которой выполните удаление объекта.

## Применение

1. Выберите одну или несколько ячеек электронной таблицы.
2. Выберите пункт меню **Правка → ![](/images/Std_Cut.svg) Вырезать**.

## Примечания

* Стандартное сочетание клавиш Ctrl+X не вызывает команду "Std\_Cut", а вместо этого использует аналогичную по своей сути функцию, но из операционной системы. Вы можете использовать это сочетание клавиш в большинстве текстовых окон и полей ввода FreeCAD, в том числе и для правки ячеек электронной таблицы.
