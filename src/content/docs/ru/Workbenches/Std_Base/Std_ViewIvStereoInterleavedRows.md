## Описание

The **Std ViewIvStereoInterleavedRows** command changes the active [3D view](/3D_view "3D view") to interleaved rows stereo view mode. To use this stereo mode a special graphics card, a special display and [glasses with polarized lenses](https://en.wikipedia.org/wiki/Polarized_3D_system) are requires.

## Применение

1. Select the **View → Stereo → ![](/images/Std_ViewIvStereoInterleavedRows.svg) Stereo interleaved Rows** option from the menu.
