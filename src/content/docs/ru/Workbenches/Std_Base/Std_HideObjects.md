## Описание

The **Std HideObjects** command hides all objects belonging to the active document in [3D views](/3D_view "3D view").

## Применение

1. Выберите пункт меню **Вид → Видимость → ![](/images/Std_HideObjects.svg) Скрыть все объекты**.
