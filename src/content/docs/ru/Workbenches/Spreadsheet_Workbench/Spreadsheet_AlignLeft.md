## Описание

The ![](/images/Spreadsheet_AlignLeft.svg) **Spreadsheet AlignLeft** tool aligns the contents of selected cells to the left.

## Применение

1. Make sure there is an active [spreadsheet](/Spreadsheet_CreateSheet "Spreadsheet CreateSheet").
2. Select one or more cells.
3. There are several ways to invoke the tool:
   * Press the ![](/images/Spreadsheet_AlignLeft.svg) [Align left](/Spreadsheet_AlignLeft "Spreadsheet AlignLeft") button.
   * Select the **Spreadsheet → Alignment → ![](/images/Spreadsheet_AlignLeft.svg) Align left** option from the menu.
