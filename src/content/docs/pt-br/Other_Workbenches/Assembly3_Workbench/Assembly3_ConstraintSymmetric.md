## Descrição

Essa ferramenta cria um vínculo entre dois objetos de uma montagem e corresponde à orientação deles. Os elementos selecionados de cada objeto ou, mais precisamente, seus sistemas de coordenadas implícitas (ICS) são usados para posicionar um objeto em outro.

## Utilização

1. Place two objects into an assembly.
2. Select one element of each object and a mirroring plane.
3. Press the ![](/images/Assembly_ConstraintSymmetric.svg) [Symmetric](/Assembly3_ConstraintSymmetric "Assembly3 ConstraintSymmetric") button.
