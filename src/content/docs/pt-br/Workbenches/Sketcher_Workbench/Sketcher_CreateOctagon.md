## Description

The ![](/images/Sketcher_CreateOctagon.svg) [Sketcher CreateOctagon](/Sketcher_CreateOctagon "Sketcher CreateOctagon") tool creates an octagon. [introduced in 1.0](/Release_notes_1.0 "Release notes 1.0"): This is the same tool as [Sketcher CreateRegularPolygon](/Sketcher_CreateRegularPolygon "Sketcher CreateRegularPolygon") but with the number of sides preset to 8.

![](/images/SketcherCreateOctagonExample.png)

## Usage

See also: [Drawing aids](/Sketcher_Workbench#Drawing_aids "Sketcher Workbench").

1. There are several ways to invoke the tool:
   * Press the ![](/images/Sketcher_CreateOctagon.svg) [Octagon](/Sketcher_CreateOctagon "Sketcher CreateOctagon") button.
   * Select the **Sketcher → Sketcher geometries → ![](/images/Sketcher_CreateOctagon.svg) Create octagon** option from the menu.
   * Use the keyboard shortcut: G then P, then 8.
2. For further steps see [Sketcher CreateRegularPolygon](/Sketcher_CreateRegularPolygon#Usage "Sketcher CreateRegularPolygon").
