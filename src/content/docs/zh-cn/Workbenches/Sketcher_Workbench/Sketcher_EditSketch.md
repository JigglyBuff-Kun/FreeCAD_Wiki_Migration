## 描述

借助此命令将使您编辑一个存在的草图。这会打开一个[草图对话框（Sketcher Dialog）](/Sketcher_Dialog "Sketcher Dialog")。

## 用法

* 在树状视图中右键某草图并选择*Edit Sketch*；或在树状视图中选择草图，并从下拉菜单中选择*Sketch → Edit Sketch*。双击草图也可达到此效果。
* 选择[离开草图（Leave Sketch）](/Sketcher_LeaveSketch "Sketcher LeaveSketch")将终止此编辑模式。
