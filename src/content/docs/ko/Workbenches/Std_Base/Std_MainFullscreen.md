## 설명

**표준 메인 전체화면(Std MainFullscreen)** 명령은 FreeCAD의 메인 창의 전체화면 모드를 전환합니다.

## 용법

1. 이 명령을 실행하는 방법은 여러 가지입니다:
   * 메뉴에서 **보기 → ![](/images/Std_MainFullscreen.svg) 전체화면** 옵션을 선택합니다.
   * 단축키를 사용합니다: Alt+F11.
