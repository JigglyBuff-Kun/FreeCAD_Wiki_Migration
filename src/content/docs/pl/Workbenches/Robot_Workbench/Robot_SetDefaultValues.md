## Description

Set the default values for way-point creation.

## Usage

1. Choose Robot → ![](/images/Robot_SetDefaultValues.svg) Set default values from the top menu, to define:

:   :   * speed
        * continuous (mode)
        * acceleration

## Notes

The program's default values are for

* speed = 1 m/s
* continuous = false
* acceleration = 1 m/s^2
