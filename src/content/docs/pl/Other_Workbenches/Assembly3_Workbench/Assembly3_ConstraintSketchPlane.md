## Description

The ![](/images/Assembly_ConstraintSketchPlane.svg) [Sketch plane](/Assembly3_ConstraintSketchPlane "Assembly3 ConstraintSketchPlane") command creates a "Sketch plane" as a container for Draft objects.

It can be used to create a skeleton sketch for 2D kinematics.

## Usage

1. Create an empty assembly container by pressing ![](/images/Assembly_New_Assembly.svg) [Create assembly](/Assembly3_CreateAssembly "Assembly3 CreateAssembly")
2. Select the newly created assembly object, and click ![](/images/Assembly_Add_Workplane.svg) [Add workplane](/Assembly3_AddWorkplane "Assembly3 AddWorkplane")
3. Press the ![](/images/Assembly_ConstraintSketchPlane.svg) [Sketch plane](/Assembly3_ConstraintSketchPlane "Assembly3 ConstraintSketchPlane") button
4. Switch to the ![](/images/Workbench_Draft.svg) [Draft Workbench](/Draft_Workbench "Draft Workbench") to add some content

:   With an existing non-empty assembly steps 1 and 2 can be skipped and any planar face element can be used as a work plane for the sketch plane.
