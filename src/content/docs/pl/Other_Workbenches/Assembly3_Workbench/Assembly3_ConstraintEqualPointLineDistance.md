## Description

The ![](/images/Assembly_ConstraintEqualPointLineDistance.svg) [Equal point line distance](/Assembly3_ConstraintEqualPointLineDistance "Assembly3 ConstraintEqualPointLineDistance") command constrains two distances, each between a 2D point and a straight 2D line, to have the same value.

## Usage

1. Select a 2D **point**.
2. Select another 2D **line**.
3. Select a 2D **point**.
4. Select another 2D **line**.
   - **Note:** Don't forget the Control/Command key!
5. Activate the ![](/images/Assembly_ConstraintEqualPointLineDistance.svg) [Equal point line distance](/Assembly3_ConstraintEqualPointLineDistance "Assembly3 ConstraintEqualPointLineDistance") command using the:
   - ![](/images/Assembly_ConstraintEqualPointLineDistance.svg) [Equal point line distance](/Assembly3_ConstraintEqualPointLineDistance "Assembly3 ConstraintEqualPointLineDistance") button.
6. Press the ![](/images/Assembly3_workbench_icon.svg) [Solve constraints](/Assembly3_ResolveConstraints "Assembly3 ResolveConstraints") or the ![](/images/Assembly_QuickSolve.svg) [Quick solve](/Assembly3_QuickSolve "Assembly3 QuickSolve") button to recompute

: : (if ![](/images/Assembly_AutoRecompute.svg) [Auto recompute](/Assembly3_AutoRecompute "Assembly3 AutoRecompute") and ![](/images/Assembly_SmartRecompute.svg) [Smart recompute](/Assembly3_SmartRecompute "Assembly3 SmartRecompute") are disabled).
