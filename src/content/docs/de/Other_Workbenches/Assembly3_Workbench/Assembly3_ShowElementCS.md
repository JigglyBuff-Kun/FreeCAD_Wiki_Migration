## Beschreibung

Der Befehl ![](/images/Assembly_ShowElementCS.svg) Element-Koordinatensystem anzeigen schaltet die Sichtbarkeit von **Element-Koordinatensystemen** (EKS) ein bzw. aus.

Ein EKS wird durch 3 Linien dargestellt, die sich im Ursprung des Elements treffen:

* eine rote Linie für die X-Achse
* eine grüne Linie für die Y-Achse
* eine blaue Linie für die Z-Achse

![](/images/Assembly3_ShowElementCS-01.png)

Element einer Fläche mit sichtbarem EKS

## Anwendung

1. Den Befehl ![](/images/Assembly_ShowElementCS.svg) **Element-Koordinatensystem anzeigen** aktivieren durch:
   * Die Schaltfläche ![](/images/Assembly_ShowElementCS.svg) Element-Koordinatensystem anzeigen.
   * Den Menüeintrag **Assembly3 → Element style → ![](/images/Assembly_ShowElementCS.svg) Element-Koordinatensystem anzeigen**.
