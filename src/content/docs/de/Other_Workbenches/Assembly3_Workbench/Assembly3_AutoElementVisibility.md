## Beschreibung

Der Befehl ![](/images/Assembly_AutoElementVis.svg) Element automatisch anzeigen schaltet die zeitweilige Hervorhebung ausgewählter **Elemente** ein bzw. aus.

Wenn aktiviert, wird das aktuell ausgewählte Element (bzw. die Elemente) hervorgehoben.   
 (Die Sichtbarkeit der Element-Koordinatensysteme (EKS) hängt von dem Umschaltstatus des Befehls ![](/images/Assembly_ShowElementCS.svg) [Element-Koordinatensystem anzeigen](/Assembly3_ShowElementCS/de "Assembly3 ShowElementCS/de") ab)

![](/images/Assembly3_AutoElementVisibility-01.png)

Hervorgehobenes Element einer Fläche mit sichtbarem EKS

## Anwendung

1. Den Befehl ![](/images/Assembly_AutoElementVis.svg) **Element automatisch anzeigen** aktivieren durch:
   * Die Schaltfläche ![](/images/Assembly_AutoElementVis.svg) Element automatisch anzeigen.
   * Den Menüeintrag **Assembly3 → Element style → ![](/images/Assembly_AutoElementVis.svg) Element automatisch anzeigen**.
