## Beschreibung

Das Werkzeug ![](/images/Curves_GeomInfo.svg) **GeomInfo** schaltet die Anzeige von Informationen zu den ausgewählten Formen ein bzw. aus.

## Anwendung

1. There are several ways to invoke the command:
   * Press the ![](/images/Curves_GeomInfo.svg) [GeomInfo](/Curves_GeomInfo "Curves GeomInfo") button.
   * Select the **Misc. → ![](/images/Curves_GeomInfo.svg) GeomInfo** option from the menu.
2. The state of the button changes and information is showm or hidden accordingly..

## Hinweise

* This command only toggles the visibility of Information, it doesn't create any object.
* On a HD display with a dark screen background the information is hardly recognizable. (It is in the upper left corner of the screen)
