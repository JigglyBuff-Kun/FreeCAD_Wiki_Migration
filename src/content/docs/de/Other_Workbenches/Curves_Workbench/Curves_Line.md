## Beschreibung

Das Werkzeug ![](/images/Curves_Line.svg) Curves Linie erstellt eine parametrische Linie zwischen zwei Knoten.

## Anwendung

1. Zwei Knoten in der [3D-Ansicht](/3D_view/de "3D view/de") auswählen.
2. Es gibt mehrere Möglichkeiten, den Befehl aufzurufen:
   * Die Schaltfläche ![](/images/Curves_Line.svg) Parametrische Linie drücken.
   * Den Menüeintrag **Kurven → ![](/images/Curves_Line.svg) Parametrische Linie** auswählen.
3. Eine parametrische Linie, ein **Line**-Objekt, wird erstellt.

## Eigenschaften

Siehe auch: [Eigenschafteneditor](/Property_editor/de "Property editor/de").

Ein **Line**-Objekt wird von einem [App FeaturePython](/App_FeaturePython/de "App FeaturePython/de")-Objekt abgeleitet und erbt alle seine Eigenschaften. Außerdem besitzt es die folgenden zusätzlichen Eigenschaften:

### Daten

Line

* Daten-Eigenschaft**Vertex1** (`LinkSub`): Der erste referenzierte Knoten.
* Daten-Eigenschaft**Vertex2** (`LinkSub`): Der zweite referenzierte Knoten.
