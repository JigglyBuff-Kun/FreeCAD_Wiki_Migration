## Beschreibung

Dieses Werkzeug erlaubt dem Anwender die Legende der Datenreihe eines bestimmten Diagramms zu aktivieren bzw. deaktivieren.

![](/images/Plot_Trigonometric_Example.png)

## Anwendung

Select the plot tab that you want to edit, and run this tool to change the state of the legend, i.e. show it if it was hidden, hide it otherwise.

Remember that the styles and labels of the data series can be edited with the [data series configuration tool](/Plot_Series "Plot Series").
