## Description

Lets you override one or more properties of a trajectory.

## Usage

1. Select trajectory or Edge2Trac-trajectory to be dressed up in the [Tree view](/Tree_view "Tree view")
2. Click on ![](/src/assets/images/Robot_TrajectoryDressUp.svg) or change to the [Task panel](/Task_panel "Task panel") and click on ![](/src/assets/images/Robot_TrajectoryDressUp.svg) Dress-up trajectory

Resulting in the ability to change values for:

- speed
- acceleration
- continous mode
- position and orientation
