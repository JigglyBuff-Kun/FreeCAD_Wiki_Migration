## Beschreibung

Das Werkzeug ![](/src/assets/images/Sketcher_RenderingOrder.svg) Sketcher RenderingReihenfolge selbst macht aktuell gar nichts. Aber im zugehörigen Menü kann die Rendering-Reihenfolge für alle Skizzen geändert werden.

Im Falle von deckungsgleichen Objekten gibt die Rendering-Reihenfolge vor, welche Elemente zuoberst liegen und daher in der [3D-Ansicht](/3D_view/de "3D view/de") zuerst ausgewählt werden.

## Anwendung

1. Auf den Nach-Unten-Pfeil rechts neben der Schaltfläche ![](/src/assets/images/Sketcher_RenderingOrder.svg) Rendering-Reihenfolge konfigurieren klicken.
2. Die Liste mit der Rendering-Reihenfolge wird geöffnet:  
   ![](/src/assets/images/Sketcher_RenderingOrder_Menu.png)
3. Einzelne Listenelemente ziehen, um die Rendering-Reihenfolge zu ändern.
4. Die (Farbe der) Schaltfläche wird passend zum ersten Element auf der Liste geändert.
5. Ein Klick außerhalb der Liste schließt die Aktion ab.
