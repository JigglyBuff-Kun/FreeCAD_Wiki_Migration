## Beschreibung

Das Werkzeug **BIM Stütze** erstellt eine **Stütze** (Column-Objekt), ein vertikales Strukturelement, eingesetzt an einer festgelegten Stelle.

## Anwendung

1. Es gibt mehrere Möglichkeiten, das Werkzeug aufzurufen:
   * Die Schaltfläche ![](/images/BIM_Column.svg) Stütze drücken.
   * Den Menüeintrag **3D/BIM → ![](/images/BIM_Column.svg) Stütze** auswählen.
   * Das Tastaturkürzel C dann O.
