## Beschreibung

Das Werkzeug **TechDraw SeiteNeuZeichnen** veranlasst eine Zeichnungsaktualisierung trotz (ggf.) gegenteiliger Einstellungen von KeepUpdated (Zeichnungs-Eigenschaft), Update with 3D (Parameter) oder Allow Page Override (Parameter).

## Anwendung

1. Wenn sich mehrere Zeichnungsblätter im Dokument befinden: wahlweise das gewünschte Blatt aktivieren oder in der [Baumansicht](/Tree_view/de "Tree view/de") auswählen.
2. Es gibt mehrere Möglichkeiten das Werkzeug aufzurufen:
   - Die Schaltfläche ![](/src/assets/images/TechDraw_RedrawPage.svg) Seite neu zeichnen drücken.
   - Den Menüentrag **TechDraw → Page → ![](/src/assets/images/TechDraw_RedrawPage.svg) Seite neu zeichnen** auswählen.
3. Wenn sich mehrere Zeichnungsblätter im Dokument befinden und noch kein Blatt ausgewählt wurde, wird das Dialogfeld **Blattauswahl** geöffnet:
   1. Das gewünschte Blatt auswählen.
   2. Die Schaltfläche OK drücken.
