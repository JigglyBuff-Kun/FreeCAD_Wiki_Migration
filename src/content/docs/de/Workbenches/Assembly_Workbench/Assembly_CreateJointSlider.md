## Beschreibung

Das Werkzeug ![](/images/Assembly_CreateJointSlider.svg) Assembly PrismatischeFührungErstellen erstellt eine prismatische Führung (auch Schubgelenk genannt) zwischen zwei ausgewählten Bauteilen, die eine geradlinige Verschiebung entlang einer einzelnen Achse ermöglicht, während sämtliche Drehungen verhindert werden.

## Anwendung

1. Wahlweise zwei geometrische Elemente zweier unterschiedlicher Formen auswählen. Andere Auswahlen werden verworfen.
2. Es gibt mehrere Möglichkeiten, das Werkzeug aufzurufen:
   * Die Schaltfläche ![](/images/Assembly_CreateJointSlider.svg) Create Slider Joint drücken.
   * Den Menüeintrag **Assembly → ![](/images/Assembly_CreateJointSlider.svg) Create Slider Joint** auswählen.
   * Das Tastaturkürzel S.
3. Der Dialog **Create Joint** wird in [Aufgaben-Tafel](/Task_panel/de "Task panel/de") geöffnet und listet die vorausgewählten Elemente.
4. Für weitere Schritte siehe [Assembly StarreVerbindungErstellen](/Assembly_CreateJointFixed/de#Anwendung "Assembly CreateJointFixed/de").
