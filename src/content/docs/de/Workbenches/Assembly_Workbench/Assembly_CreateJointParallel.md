## Beschreibung

Das Werkzeug ![](/images/Assembly_CreateJointParallel.svg) Assembly ErzeugeParallelVerbindung erstellt eine Parallelführung zwischen zwei ausgewählten Bauteilen, dabei werden die Z-Achsen der ausgewählten Koordinatensysteme parallel ausgerichtet.

## Anwendung

1. Es gibt mehrere Möglichkeiten, das Werkzeug aufzurufen:
   * Die Schaltfläche ![](/images/Assembly_CreateJointParallel.svg) [Create Parallel Joint](/Assembly_CreateJointParallel "Assembly CreateJointParallel") drücken.
   * Den Menüeintrag **Assembly → ![](/images/Assembly_CreateJointParallel.svg) Create Parallel Joint** auswählen.
   * Das Tastaturkürzel N.
2. ...
