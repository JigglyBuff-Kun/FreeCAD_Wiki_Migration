## Usage

1. Switch to the ![](/src/assets/images/Design456_workbench_icon.svg) [Design456](/Design456_Workbench "Design456 Workbench") workbench (install from ![](/src/assets/images/Std_AddonMgr.svg) [Addon Manager](/Std_AddonMgr "Std AddonMgr") is necessary, if not previously installed)
2. Select a face from a 3D object
3. Press the ![](/src/assets/images/Design456_ExtrudeFace.svg) Extrude Face command. A copy of the face will be created and the command ask you for the length of the extrusion. Later the extruded face will be created on the same direction of the face.

## Limitations

Works on 3D objects.
