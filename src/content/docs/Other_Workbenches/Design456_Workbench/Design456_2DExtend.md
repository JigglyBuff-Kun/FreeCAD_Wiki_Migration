## Usage

1. Switch to the ![](/images/Design456_workbench_icon.svg) [Design456](/Design456_Workbench "Design456 Workbench") workbench (install from ![](/images/Std_AddonMgr.svg) [Addon Manager](/Std_AddonMgr "Std AddonMgr") is necessary, if not previously installed)
2. The aim of this tool is to easily Extend an edge or a wire.

3. Invoke the ExtendLine tool several ways:
   - Press the ![](/images/2D_ExtendLine.svg) button
   - Use the **Design456_2Ddrawing → Extend Line** entry in the Design456 menu

## Limitations

At the moment, the command extend only at the end. This must be improved to do line extend in all directions.
