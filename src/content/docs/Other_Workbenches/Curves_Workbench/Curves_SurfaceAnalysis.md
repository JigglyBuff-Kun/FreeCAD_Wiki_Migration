## Description

The [File:Curves SurfaceAnalysis.svg](/index.php?title=Special:Upload&wpDestFile=Curves_SurfaceAnalysis.svg "File:Curves SurfaceAnalysis.svg") **Curves SurfaceAnalysis** tool creates visual analysis maps on selected shapes.

## Usage

1. Select one or more sub shapes.
2. There are several ways to invoke the tool:
   - Press the ![](/src/assets/images/Curves_ZebraTool.svg) Title button.
   - Select the **Surfaces → ![](/src/assets/images/Curves_ZebraTool.svg) Title** option from the menu.
3. A SurfaceAnalysis object is created.
