## Description

The ![](/src/assets/images/Curves_ExtractSubshape.svg) **Curves ExtractSubshape** tool creates non-parametric copies of selected sub shapes.

## Usage

1. Select one or more conical or cylindrical faces.
2. There are several ways to invoke the tool:
   - Press the ![](/src/assets/images/Curves_ExtractSubshape.svg) Extract subshape button.
   - Select the **Misc. → ![](/src/assets/images/Curves_ExtractSubshape.svg) Extract subshape** option from the menu.
3. A copy of each selected sub shape is created.
