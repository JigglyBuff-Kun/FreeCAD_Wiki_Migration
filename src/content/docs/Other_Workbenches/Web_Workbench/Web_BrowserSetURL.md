## Description

Use the ![](/src/assets/images/Web_BrowserSetURL.svg) **Web BrowserSetURL** command to change the URL of the active Web tab.

## Usage

1. Make sure a Web tab is active, use the ![](/src/assets/images/Web_OpenWebsite.svg) [Web OpenWebsite](/Web_OpenWebsite "Web OpenWebsite") command to create one if required.
2. Press the ![](/src/assets/images/Web_BrowserSetURL.svg) [Set URL](/Web_BrowserSetURL "Web BrowserSetURL") button.
3. A prompt line opens with the current URL.
4. Edit the URL and press Enter to finish.
5. The new URL is loaded.
